## 1. 固定翼与多旋翼的动力学差异

### 1.1 基本对比

| 特性 | 固定翼 (Fixed-Wing) | 多旋翼 (Multirotor) |
|------|---------------------|---------------------|
| 升力来源 | 机翼气动升力 | 旋翼推力 |
| 悬停能力 | 不能 (失速速度 > 0) | 能 (推力 = 重力) |
| 能量效率 | 高 (滑翔比可达 15:1) | 低 (悬停功耗大) |
| 速度范围 | 20–100 m/s | 0–20 m/s |
| 航程 | 长 (数十至数百 km) | 短 (数 km) |
| 控制面 | 副翼、升降舵、方向舵 | 各旋翼转速差 |
| 动力学复杂度 | 高 (气动耦合) | 中等 |
| 失速/失速特性 | 有 | 无 |

### 1.2 运动方式的根本区别

```
┌─────────────────────────────────────────────────────────┐
│                   固定翼飞行原理                          │
│                                                         │
│         升力 L ↑                                         │
│              │    ╱╲                                     │
│              │   ╱  ╲  机翼截面                          │
│              │  ╱    ╲                                   │
│    推力 T ──►├─────────► 阻力 D                          │
│              │                                          │
│              │    重力 W ↓                               │
│                                                         │
│  稳态飞行: L = W, T = D                                 │
│  需要维持最小速度以产生足够升力                            │
└─────────────────────────────────────────────────────────┘
```

## 2. 气动力与气动力矩

### 2.1 气动力

固定翼无人机受到的三个主要气动力分量：

**升力 (Lift)：**

$$
L = \frac{1}{2} \rho V^2 S C_L
$$

**阻力 (Drag)：**

$$
D = \frac{1}{2} \rho V^2 S C_D
$$

**侧力 (Side Force)：**

$$
Y = \frac{1}{2} \rho V^2 S C_Y
$$

其中：
- $\rho$：空气密度 (kg/m³)
- $V$：空速 (m/s)
- $S$：机翼参考面积 (m²)
- $C_L, C_D, C_Y$：无量纲气动系数

### 2.2 升力系数

升力系数是迎角 (angle of attack) $\alpha$ 和升降舵偏转 $\delta_e$ 的函数：

$$
C_L = C_{L_0} + C_{L_\alpha} \alpha + C_{L_{\delta_e}} \delta_e
$$

**典型参数：**

| 参数 | 符号 | 典型值 | 说明 |
|------|------|--------|------|
| 零迎角升力系数 | $C_{L_0}$ | 0.2–0.5 | 取决于翼型弯度 |
| 升力线斜率 | $C_{L_\alpha}$ | 4.5–6.0 /rad | 取决于展弦比 |
| 升降舵升力效率 | $C_{L_{\delta_e}}$ | 0.3–0.8 /rad | 取决于舵面大小 |

**非线性升力模型 (失速区)：**

```
  C_L
   ▲
   │           ╱───── 线性区
   │         ╱
   │       ╱    失速
   │     ╱   ╲
   │   ╱       ╲
   │ ╱           ╲───── 失速后
   │╱
   └──────────────────────► α
   0°     α_stall    20°
```

### 2.3 阻力系数

阻力分为**零升阻力**和**诱导阻力**：

$$
C_D = C_{D_0} + C_{D_i} = C_{D_0} + \frac{C_L^2}{\pi e AR}
$$

其中：
- $C_{D_0}$：零升阻力系数（摩擦 + 形状阻力）
- $e$：Oswald 效率因子 (0.7–0.95)
- $AR = b^2 / S$：展弦比

**极曲线 (Drag Polar)：**

```
  C_D
   ▲
   │                    ╱
   │                  ╱
   │                ╱
   │              ╱
   │            ╱  抛物线极曲线
   │          ╱
   │        ╱
   │  C_D0 ╱
   └──────────────────────► C_L
        0    C_L_opt  C_L_max
```

### 2.4 气动力矩

**滚转力矩 (Roll Moment)：**

$$
L_{aero} = \frac{1}{2} \rho V^2 S b \left( C_{l_\beta} \beta + C_{l_{\delta_a}} \delta_a + C_{l_{\delta_r}} \delta_r + \frac{b}{2V} C_{l_p} p \right)
$$

**俯仰力矩 (Pitch Moment)：**

$$
M_{aero} = \frac{1}{2} \rho V^2 S \bar{c} \left( C_{m_0} + C_{m_\alpha} \alpha + C_{m_{\delta_e}} \delta_e + \frac{\bar{c}}{2V} C_{m_q} q \right)
$$

**偏航力矩 (Yaw Moment)：**

$$
N_{aero} = \frac{1}{2} \rho V^2 S b \left( C_{n_\beta} \beta + C_{n_{\delta_r}} \delta_r + \frac{b}{2V} C_{n_r} r \right)
$$

其中：
- $b$：翼展 (m)
- $\bar{c}$：平均气动弦长 (m)
- $\beta$：侧滑角 (rad)
- $\delta_a, \delta_e, \delta_r$：副翼、升降舵、方向舵偏转 (rad)
- $p, q, r$：机体系角速度分量 (rad/s)

## 4. 推进系统

### 4.1 螺旋桨模型

**推力系数和功率系数：**

$$
T = C_T \rho n^2 D^4
$$

$$
P = C_P \rho n^3 D^5
$$

其中 $n$ 为转速 (rev/s)，$D$ 为桨径 (m)。

**进速比 (Advance Ratio)：**

$$
J = \frac{V}{nD}
$$

$C_T$ 和 $C_P$ 是 $J$ 的函数，通过实验数据或经验公式获得。

### 4.2 发动机模型

对于活塞发动机：

$$
T = \frac{\eta_p P_{shaft}}{V}
$$

其中 $\eta_p$ 为螺旋桨效率，$P_{shaft}$ 为轴功率。

### 4.3 Simulink 推进系统模型

```
┌──────────────────────────────────────────────────────┐
│               推进系统 Simulink 模型                    │
│                                                      │
│  V ──►[J=V/nD]──►[CT(J)]──►[T=CT*ρ*n²*D⁴]──► Thrust│
│                    ▲                                 │
│  n ────────────────┘                                 │
│                                                      │
│  V ──►[J=V/nD]──►[CP(J)]──►[P=CP*ρ*n³*D⁵]──► Power │
```

## 5. 配平 (Trim Condition)

### 5.1 配平概念

配平是指找到一组控制输入和状态，使得所有力和力矩平衡（稳态飞行条件）。

**配平方程：**

$$
\sum \mathbf{F} = 0, \quad \sum \mathbf{M} = 0
$$

对于定常直线飞行：

$$
L = W \cos\gamma, \quad T = D + W\sin\gamma
$$

$$
M = 0, \quad L_{roll} = 0, \quad N = 0
$$

### 5.2 配平变量

| 变量 | 说明 |
|------|------|
| $\alpha_{trim}$ | 配平迎角 |
| $\delta_{e,trim}$ | 配平升降舵偏转 |
| $\delta_{t,trim}$ | 配平油门 |
| $V_{trim}$ | 配平空速 |

### 5.3 MATLAB 配平函数

```matlab
function [x_trim, u_trim] = trim_fixed_wing(params, V_trim, gamma_trim)
% 固定翼配平计算
% 使用 fmincon 求解非线性优化问题

    % 初始猜测
    x0 = [0; 0; -100;   % 位置 (NED)
          V_trim; 0; 0;  % 速度
          0; 0.05; 0;    % 姿态 (phi, theta, alpha)
          0; 0; 0];      % 角速度

    u0 = [0.05;          % 升降舵
          0;             % 副翼
          0;             % 方向舵
          0.6];          % 油门

    % 优化目标：力和力矩平衡
    cost_fn = @(xu) norm(dynamics_fixed_wing(xu(1:12), xu(13:16), params))^2;

    % 约束
    lb = [-inf*ones(12,1); -0.5; -0.5; -0.5; 0];  % 控制下限
    ub = [ inf*ones(12,1);  0.5;  0.5;  0.5; 1];   % 控制上限

    % 求解
    options = optimoptions('fmincon', 'Display', 'iter', ...
                           'Algorithm', 'sqp', 'MaxIterations', 1000);
    result = fmincon(cost_fn, [x0; u0], [], [], [], [], lb, ub, [], options);

    x_trim = result(1:12);
    u_trim = result(13:16);
end
```

## 6. 纵向与横侧向动力学

### 6.1 纵向动力学

纵向运动包括**前飞速度** ($u$)、**垂直速度** ($w$)、**俯仰角** ($\theta$) 和**俯仰率** ($q$)。

**纵向小扰动线性化方程：**

$$
\begin{bmatrix} \Delta\dot{u} \\ \Delta\dot{w} \\ \Delta\dot{q} \\ \Delta\dot{\theta} \end{bmatrix} = \mathbf{A}_{long} \begin{bmatrix} \Delta u \\ \Delta w \\ \Delta q \\ \Delta\theta \end{bmatrix} + \mathbf{B}_{long} \begin{bmatrix} \Delta\delta_e \\ \Delta\delta_t \end{bmatrix}
$$

### 6.2 纵向模态

| 模态 | 特征值特征 | 周期 | 阻尼比 | 物理含义 |
|------|-----------|------|--------|----------|
| **短周期** (Short Period) | 大负实部或复数 | 1–5 s | 0.3–0.7 | 快速迎角变化 |
| **长周期/浮沉** (Phugoid) | 小实部 + 大虚部 | 30–100 s | 0.01–0.1 | 速度与高度交替交换 |

**浮沉模态 (Phugoid)：**

```
  高度 h
   ▲
   │    ╱╲        ╱╲        ╱╲
   │  ╱    ╲    ╱    ╲    ╱    ╲
   │╱        ╲╱        ╲╱        ╲─── 缓慢衰减
   └──────────────────────────────────► 时间
   │  ◄── 长周期 T ≈ 50s ──►│
   │      极低阻尼 ζ ≈ 0.05  │
```

**短周期模态：**

```
  迎角 α
   ▲
   │ ╱╲
   │╱  ╲
   │     ╲─── 快速收敛
   │
   └──────────────────────────► 时间
   │◄─短周期─►│
   │ T ≈ 2s   │
```

### 6.3 横侧向动力学

横侧向运动包括**侧滑角** ($\beta$)、**滚转率** ($p$)、**偏航率** ($r$) 和**滚转角** ($\phi$)。

**横侧向小扰动线性化方程：**

$$
\begin{bmatrix} \Delta\dot{\beta} \\ \Delta\dot{p} \\ \Delta\dot{r} \\ \Delta\dot{\phi} \end{bmatrix} = \mathbf{A}_{lat} \begin{bmatrix} \Delta\beta \\ \Delta p \\ \Delta r \\ \Delta\phi \end{bmatrix} + \mathbf{B}_{lat} \begin{bmatrix} \Delta\delta_a \\ \Delta\delta_r \end{bmatrix}
$$

### 6.4 横侧向模态

| 模态 | 特征值特征 | 周期 | 物理含义 |
|------|-----------|------|----------|
| **荷兰滚** (Dutch Roll) | 复数 | 1–5 s | 侧滑与偏航耦合振荡 |
| **螺旋模态** (Spiral) | 小正/负实数 | > 20 s | 缓慢滚转偏离 |
| **滚转衰减** (Roll Subsidence) | 大负实数 | < 1 s | 快速滚转收敛 |

```
  β (侧滑角)
   ▲        荷兰滚
   │  ╱╲    ╱╲    ╱╲
   │╱    ╲╱    ╲╱    ╲──── 快速衰减
   │
   └──────────────────────────► 时间
```

## 7. 六自由度动力学方程

### 7.1 完整非线性方程

**力方程 (机体系)：**

$$
m(\dot{u} + qw - rv) = F_x
$$

$$
m(\dot{v} + ru - pw) = F_y
$$

$$
m(\dot{w} + pv - qu) = F_z
$$

**力矩方程 (机体系)：**

$$
I_x \dot{p} - (I_y - I_z)qr = L_{total}
$$

$$
I_y \dot{q} - (I_z - I_x)pr = M_{total}
$$

$$
I_z \dot{r} - (I_x - I_y)pq = N_{total}
$$

其中 $F_x, F_y, F_z$ 包含重力、气动力和推力在机体系下的分量。

### 7.2 力的分解

$$
\begin{bmatrix} F_x \\ F_y \\ F_z \end{bmatrix}_{body} = \mathbf{R}_{wind \to body} \begin{bmatrix} -D \\ Y \\ -L \end{bmatrix} + \mathbf{R}_{earth \to body} \begin{bmatrix} 0 \\ 0 \\ mg \end{bmatrix} + \begin{bmatrix} T \\ 0 \\ 0 \end{bmatrix}
$$

### 8.3 气动力计算子系统

```matlab
function [F_aero, M_aero] = aero_forces(V, alpha, beta, p, q, r, ...
                                         delta_e, delta_a, delta_r, params)
    % 动压
    qbar = 0.5 * params.rho * V^2;

    % 气动力系数
    CL = params.CL0 + params.CLa*alpha + params.CLde*delta_e;
    CD = params.CD0 + CL^2 / (pi * params.e * params.AR);
    CY = params.CYb*beta + params.CYdr*delta_r;

    % 气动力 (风轴系)
    L = qbar * params.S * CL;
    D = qbar * params.S * CD;
    Y = qbar * params.S * CY;

    % 转换到机体系
    F_aero = [cos(alpha)*(-D) + sin(alpha)*(-L);
              Y;
              sin(alpha)*(-D) + cos(alpha)*(-L)];

    % 气动力矩
    b = params.b;  % 翼展
    c = params.c;  % 平均气动弦

    Cl = params.Clb*beta + params.Clda*delta_a + params.Cldr*delta_r + ...
         (b/(2*V))*params.Clp*p;
    Cm = params.Cm0 + params.Cma*alpha + params.Cmde*delta_e + ...
         (c/(2*V))*params.Cmq*q;
    Cn = params.Cnb*beta + params.Cndr*delta_r + ...
         (b/(2*V))*params.Cnr*r;

    M_aero = qbar * params.S * [b*Cl; c*Cm; b*Cn];
end
```

## 思考题

1. **为什么固定翼无人机不能像多旋翼那样悬停？从升力公式的角度解释。**

<details><summary>参考答案</summary>
根据升力公式 $L = \frac{1}{2}\rho V^2 S C_L$，升力与空速 $V$ 的平方成正比。当 $V = 0$ 时，$L = 0$，无论 $C_L$ 多大都无法产生升力。实际上，当空速低于失速速度时，机翼上表面气流分离，$C_L$ 急剧下降，升力迅速减小。多旋翼的升力来自旋翼旋转，即使地面静止也能产生推力（旋翼始终在"旋转前进"），因此可以悬停。VTOL 固定翼（如倾转旋翼）通过在悬停时切换到旋翼模式来解决这一问题。
</details>

2. **浮沉模态 (Phugoid) 的物理机制是什么？为什么它的阻尼比通常很低？**

<details><summary>参考答案</summary>
浮沉模态是动能和势能之间的缓慢交换过程。当飞机稍微上仰时，高度增加但速度减小（动能→势能）；随后由于速度不足，升力减小，飞机下降，速度增加（势能→动能），如此交替。阻尼低是因为浮沉模态中迎角几乎不变（$\Delta\alpha ≈ 0$），气动阻力变化很小，而能量耗散主要靠阻力。对于无动力飞机，浮沉模态近乎无阻尼（纯正弦振荡）；有动力时，推力随速度变化提供少量阻尼。这也是为什么自动着陆时需要特别处理浮沉模态。
</details>

3. **在 Simulink 中实现气动力计算模块时，为什么通常将风轴系和机体系分开处理？**

<details><summary>参考答案</summary>
气动力系数（$C_L, C_D, C_Y$）是在**风轴系** (wind frame) 中定义的：升力垂直于来流方向，阻力平行于来流方向。但六自由度动力学方程在**机体系** (body frame) 中求解。因此需要两步：(1) 在风轴系中根据 $\alpha, \beta$ 计算气动力；(2) 通过旋转矩阵 $\mathbf{R}_{wind \to body}$ 转换到机体系。这样做的好处是气动力系数可以直接使用风洞数据或 CFD 结果，不需要每次都重新定义参考系。
</details>

4. **荷兰滚 (Dutch Roll) 模态的名称由来是什么？它对飞行品质有什么影响？**

<details><summary>参考答案</summary>
"荷兰滚"得名于荷兰传统木鞋舞蹈中身体左右摇摆+旋转的动作。飞机的荷兰滚模态是侧滑角 $\beta$ 和偏航角 $\psi$ 的耦合振荡：飞机向一侧侧滑时产生偏航力矩使其转向，但惯性使其超过平衡点后向另一侧侧滑，如此交替。同时滚转角也跟随变化。对飞行品质的影响：(1) 中等阻尼的荷兰滚是可以接受的，提供机动灵活性；(2) 低阻尼的荷兰滚严重影响乘客舒适性和飞行员任务执行；(3) MIL-STD-1797 要求 $\zeta_{DR} > 0.08$，$\omega_{n,DR} > 0.4$ rad/s。方向舵面积和垂尾设计对荷兰滚阻尼有决定性影响。
</details>

5. **固定翼无人机的配平 (Trim) 问题可以转化为什么数学问题？为什么要用数值方法求解？**

<details><summary>参考答案</summary>
配平问题可以转化为**非线性方程组求解**或**约束优化问题**：找到一组状态 $\mathbf{x}_{trim}$ 和控制输入 $\mathbf{u}_{trim}$，使得所有合力和合力矩为零，同时满足一些约束（如高度保持、指定空速）。这等价于最小化 $f(\mathbf{x}, \mathbf{u}) = \|\sum \mathbf{F}\|^2 + \|\sum \mathbf{M}\|^2$。需要数值方法是因为：(1) 气动力系数包含三角函数和查表函数，方程组是非线性的；(2) 通常欠定（自由度多于方程数），需要额外约束指定飞行条件；(3) 解析求解需要大量简化假设，精度不足。MATLAB 的 `fmincon` 或 `fsolve` 可以有效求解此类问题。
</details>
