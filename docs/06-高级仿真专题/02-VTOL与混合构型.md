## 1. VTOL 概述

### 1.1 什么是 VTOL？

VTOL（Vertical Takeoff and Landing，垂直起降）无人机结合了多旋翼的**垂直起降**能力和固定翼的**高效巡航**能力，是当前 UAV 领域的重要发展方向。

```
┌─────────────────────────────────────────────────────────────┐
│                 VTOL 的设计目标                               │
│                                                             │
│  多旋翼优势          固定翼优势                               │
│  ┌──────────┐       ┌──────────┐                            │
│  │ 垂直起降 │       │ 高能效   │                            │
│  │ 空中悬停 │       │ 高速度   │                            │
│  │ 精确着陆 │       │ 长航程   │                            │
│  └──────────┘       └──────────┘                            │
│        │                  │                                 │
│        └────────┬─────────┘                                 │
│                 ▼                                            │
│         ┌──────────────┐                                    │
│         │   VTOL UAV   │                                    │
│         │ 两者兼顾     │                                    │
│         └──────────────┘                                    │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 VTOL 为何重要？

| 应用场景 | 多旋翼的局限 | VTOL 优势 |
|----------|-------------|-----------|
| 长距离巡检 | 航程短 (5-15 km) | 航程可达 100+ km |
| 海上作业 | 无法在船上起降平台起降 | 可在小平台起降 |
| 山区物流 | 能效低 | 能效高，载重大 |
| 应急通信 | 续航时间短 | 续航可达 2-4 小时 |

## 2. VTOL 构型分类

### 2.1 倾转旋翼 (Tilt-Rotor)

```
  悬停模式                   巡航模式
  ┌─────┐                   ┌─────┐
  │旋翼 │ ← 垂直            │     │  旋翼 → 水平
  │ ↕   │                   │ ─── │
  │     │                   │     │
  └─────┘                   └─────┘
  前进方向 →                 前进方向 →
```

- 旋翼安装在可倾转的短舱上
- 悬停时旋翼朝上，巡航时旋翼朝前
- 代表：V-22 Osprey 的缩小版、DJI 飞行汽车概念

### 2.2 倾转机翼 (Tilt-Wing)

```
  悬停模式                   巡航模式
  ┌─────┐                   ┌─────┐
  │整机 │ ← 整个机翼        │ 整机│  机翼 → 水平
  │翼 ↕ │   连同螺旋桨      │ ───│
  │     │   一起倾转         │     │
  └─────┘                   └─────┘
```

- 整个机翼（而非旋翼短舱）倾转
- 翼面在悬停时垂直，减少阻力
- 结构较简单，但过渡段控制更复杂

### 2.3 尾座式 (Tail-Sitter)

```
  悬停模式                   巡航模式
  ┌─────┐                   ┌─────┐
  │     │                   │     │
  │     │ ← 垂直            │ ─── │ ← 水平
  │  ▲  │   (像火箭一样)     │  ▲  │   (正常飞行)
  │  │  │                   │  │  │
  └──┴──┘                   └──┴──┘
```

- 整个飞机绕纵轴旋转 90°
- 悬停时垂直站立，巡航时水平飞行
- 结构最简单，但过渡段姿态变化大

### 2.4 多旋翼 + 固定翼混合 (Hybrid)

```
  ┌───────────────────────────────────┐
  │     ○   ┌─────────────┐   ○     │
  │   (前旋翼)  固定机翼   (前旋翼)   │
  │     ○   │             │   ○     │
  │          │     机身    │         │
  │     ○   │             │   ○     │
  │   (后旋翼)           (后旋翼)    │
  │          └─────────────┘         │
  │                                  │
  │  巡航时：旋翼停转，仅用机翼       │
  │  悬停时：旋翼工作，机翼产生阻力   │
  └───────────────────────────────────┘
```

- 悬停时使用多旋翼，巡航时切换到固定翼
- 部分旋翼在巡航时停转或折叠
- 代表：DJI FlyCart、翼龙系列

## 4. 过渡飞行建模

### 4.1 过渡段的挑战

过渡飞行是 VTOL 建模中最复杂的部分：

```
速度 (m/s)
  ▲
  │                              ┌─── 巡航
  │                         ╱────┘
  │                    ╱───┘
  │               ╱───┘   过渡段
  │          ╱───┘        (最复杂)
  │     ╱───┘
  │ ╱───┘
  │╱
  └──────────────────────────────► 时间
  悬停        过渡          巡航
```

**关键挑战：**
- 气动特性随倾转角连续变化
- 推力方向改变导致升力/推力分配变化
- 控制策略需要无缝切换
- 过渡段可能经过不稳定的迎角区间

### 4.2 过渡段气动干扰

| 干扰源 | 影响 |
|--------|------|
| 旋翼下洗流吹过机翼 | 机翼升力变化，增加阻力 |
| 机翼前缘涡 | 悬停时旋翼洗流导致非定常涡 |
| 旋翼-机翼间距 | 间距小则干扰大 |
| 机身遮蔽 | 部分旋翼推力被机身遮挡 |

### 4.3 过渡段气动力模型

```matlab
function [F, M] = vol_transition_aero(V, alpha, tilt_angle, omega_rotors, params)
% VTOL 过渡飞行气动力计算
% tilt_angle: 旋翼倾转角 (0=悬停, 90=巡航)

    qbar = 0.5 * params.rho * V^2;

    % 旋翼推力 (随倾转角分解)
    T_rotor = params.Ct * params.rho * omega_rotor.^2 * params.D^4;
    T_x = sum(T_rotor) * cosd(tilt_angle);  % 前向推力分量
    T_z = sum(T_rotor) * sind(tilt_angle);  % 垂直推力分量

    % 机翼气动力 (仅当有来流时有效)
    if V > 1.0  % 避免除零
        CL = params.CL0 + params.CLa * alpha;
        CD = params.CD0 + CL^2 / (pi * params.e * params.AR);
        L = qbar * params.S * CL;
        D = qbar * params.S * CD;
    else
        L = 0;
        D = 0;
    end

    % 旋翼下洗流对机翼的影响
    % (简化模型: 下洗速度降低有效迎角)
    v_downwash = sqrt(max(T_z / (2 * params.rho * params.S_rotor), 0));
    alpha_eff = alpha - atan2(v_downwash, max(V, 0.1));
    % 重新计算有下洗影响的气动力...

    F = [T_x - D * cos(alpha) - L * sin(alpha);
         0;
         T_z + L * cos(alpha) - D * sin(alpha) - params.m * params.g];

    % 力矩计算
    M = [...];  % 包含气动力矩和旋翼力矩
end
```

## 5. 控制分配

### 5.1 混合构型的控制挑战

VTOL 的执行器数量通常**多于**可控自由度，需要**控制分配** (Control Allocation)。

**可控自由度：** 4 个（升力、前向力、俯仰、滚转/偏航）

**可能的执行器：**
- 多个旋翼 (4-8 个)
- 倾转机构 (1-4 个)
- 固定翼控制面 (副翼、升降舵、方向舵)

### 5.2 控制分配矩阵

$$
\begin{bmatrix} F_z \\ F_x \\ M_x \\ M_y \\ M_z \end{bmatrix} = \mathbf{B}(\gamma) \begin{bmatrix} \omega_1 \\ \vdots \\ \omega_n \\ \delta_a \\ \delta_e \\ \delta_r \\ \gamma_1 \\ \vdots \\ \gamma_m \end{bmatrix}
$$

其中 $\gamma$ 为倾转角，$\mathbf{B}(\gamma)$ 是随倾转角变化的效率矩阵。

### 5.3 Simulink 控制分配策略

```
┌───────────────────────────────────────────────────────────┐
│                VTOL 控制分配 Simulink 模型                 │
│                                                           │
│  ┌─────────┐                                              │
│  │飞行模式 │──► 模式选择 ──────────────────────┐          │
│  │状态机   │    (悬停/过渡/巡航)               │          │
│  └─────────┘                                   │          │
│                                                ▼          │
│  ┌─────────┐   ┌──────────────────────────────────┐       │
│  │姿态控制 │──►│       控制分配模块               │       │
│  │器输出   │   │  悬停: 全旋翼分配               │       │
│  │(力/力矩)│   │  过渡: 混合分配 (旋翼+舵面)     │       │
│  └─────────┘   │  巡航: 主要舵面分配             │       │
│                └──────────┬───────────────────────┘       │
│                           │                               │
│              ┌────────────┼────────────┐                  │
│              ▼            ▼            ▼                  │
│         ┌────────┐  ┌────────┐  ┌────────┐              │
│         │旋翼命令│  │倾转命令│  │舵面命令│              │
│         │ω₁..ωₙ │  │γ₁..γₘ │  │δa,δe,δr│              │
│         └────────┘  └────────┘  └────────┘              │
└───────────────────────────────────────────────────────────┘
```

### 5.4 模式切换逻辑

```matlab
function mode = vtol_mode_selector(V_airspeed, h_agl)
% VTOL 飞行模式选择

    if V_airspeed < 5
        mode = 'HOVER';         % 悬停模式
    elseif V_airspeed < 15
        mode = 'TRANSITION';    % 过渡模式
    else
        mode = 'CRUISE';        % 巡航模式
    end

    % 安全优先：低空强制悬停模式
    if h_agl < 2
        mode = 'HOVER';
    end
end
```

## 6. Simulink 建模策略

### 6.1 分阶段建模方法

```
阶段1: 悬停模型验证 ──────────────────────────────────►
         (使用多旋翼模型，验证悬停性能)

阶段2: 巡航模型验证 ──────────────────────────────────►
         (使用固定翼模型，验证巡航性能)

阶段3: 过渡模型构建 ──────────────────────────────────►
         (连接悬停和巡航，实现过渡段)

阶段4: 完整模型集成 ──────────────────────────────────►
         (模式切换 + 控制分配 + 故障处理)
```

## 思考题

1. **为什么过渡飞行是 VTOL 设计中最具挑战性的阶段？从气动力和控制两个角度分析。**

<details><summary>参考答案</summary>
从气动力角度：过渡飞行中，旋翼倾转角从 0° 变到 90°，推力方向连续变化，同时空速从 0 增加到巡航速度。在低速区间（约 5-15 m/s），机翼处于低效区（升力不足），旋翼推力分解后垂直分量也不足，形成"升力缺口"。此外旋翼下洗流吹过机翼表面，引起非定常气动效应，难以精确建模。从控制角度：不同飞行模式的控制策略差异很大（悬停用旋翼差速，巡航用舵面），过渡段需要平滑切换且不能出现不稳定的中间状态。执行器的控制分配矩阵随倾转角连续变化，需要实时求解。
</details>

2. **倾转旋翼和混合式 VTOL 各有什么优缺点？在物流配送场景中应选择哪种？**

<details><summary>参考答案</summary>
倾转旋翼优势：巡航效率高（旋翼作为螺旋桨使用），速度范围宽。缺点：倾转机构增加了结构复杂度和重量，故障模式多（倾转卡死）。混合式 VTOL 优势：结构简单可靠（无倾转机构），悬停和巡航各自独立优化。缺点：巡航时悬停旋翼成为死重和阻力源，效率略低。在物流配送场景中推荐**混合式 VTOL**，因为：(1) 载荷能力更重要，混合式的结构简单允许更大的有效载荷；(2) 物流飞行路线固定，不需要极端速度范围；(3) 可靠性要求高，避免倾转机构故障风险；(4) 维护成本低。
</details>

3. **如何设计 VTOL 的控制分配算法，使其在过渡段同时使用旋翼和舵面？**

<details><summary>参考答案</summary>
可以采用**加权伪逆法**：定义旋翼和舵面的效率矩阵 $\mathbf{B}_{rotor}$ 和 $\mathbf{B}_{surface}$，根据倾转角 $\gamma$ 加权组合：$\mathbf{B}_{total}(\gamma) = w(\gamma)\mathbf{B}_{rotor} + (1-w(\gamma))\mathbf{B}_{surface}$，其中 $w(\gamma)$ 是从 1（悬停）到 0（巡航）的连续权重函数。控制分配求解：$\mathbf{u} = \mathbf{B}_{total}^{\dagger} \boldsymbol{\tau}_{desired}$，其中 $\boldsymbol{\tau}_{desired}$ 是期望的合力/力矩，$\mathbf{B}^{\dagger}$ 是加权伪逆。还需要考虑执行器饱和约束，使用受限最小二乘或二次规划求解。
</details>

4. **在 Simulink 中，如何用 Stateflow 实现 VTOL 的模式切换状态机？**

<details><summary>参考答案</summary>
Stateflow 状态机设计：定义三个主状态（HOVER、TRANSITION、CRUISE），每个状态内部包含子状态（如 TRANSITION 内部分为 ACCELERATE 和 DECELERATE）。转换条件基于空速阈值：HOVER→TRANSITION（V > 3 m/s 且收到巡航指令），TRANSITION→CRUISE（V > 15 m/s 且倾转到位），CRUISE→TRANSITION（V < 13 m/s 且收到降落指令），TRANSITION→HOVER（V < 5 m/s 且倾转到垂直）。在每个状态中激活对应的控制器和控制分配策略。使用历史节点 (History Junction) 记住前一状态，避免紧急模式切换时丢失状态信息。
</details>

5. **尾座式 VTOL 在过渡飞行中为什么特别容易失稳？如何缓解？**

<details><summary>参考答案</summary>
尾座式 VTOL 在过渡中需要将整个飞机从垂直旋转到水平，过程中迎角从 90°（悬停）变化到接近 0°（巡航）。在 45° 左右的迎角下，机翼和机身处于高阻力、低升力的状态，且气动中心快速移动，导致静稳定性突然变化。此外，尾座式在悬停时靠旋翼推力矢量控制姿态，但在中等迎角下舵面效率很低，形成控制"死区"。缓解措施：(1) 使用推力矢量控制（倾转旋翼或导流片）补充舵面效率；(2) 采用自适应控制律，在过渡段自动调整增益；(3) 增加气动面面积提高中等迎角下的控制能力；(4) 使用前馈控制补偿已知的气动非线性。
</details>
