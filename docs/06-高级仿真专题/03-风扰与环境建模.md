## 1. 风扰建模的重要性

### 1.1 为什么需要风扰模型？

| 无风扰模型 | 有风扰模型 |
|-----------|-----------|
| 控制器在仿真中表现完美 | 控制器在真实环境中可能失效 |
| 位置跟踪精度被高估 | 能评估控制器鲁棒性 |
| 无法评估抗风能力 | 可量化最大抗风等级 |
| 无法发现潜在振荡 | 能发现风致振荡问题 |

## 2. 恒定风模型

### 2.1 基本模型

恒定风是最简单的风模型，用风速大小 $V_w$ 和风向角 $\chi_w$ 描述：

$$
\mathbf{v}_{wind} = V_w \begin{bmatrix} \cos\chi_w \\ \sin\chi_w \\ 0 \end{bmatrix}
$$

**空速与地速的关系：**

$$
\mathbf{v}_{air} = \mathbf{v}_{ground} - \mathbf{v}_{wind}
$$

其中 $\mathbf{v}_{air}$ 为空速（飞机相对于空气的速度），$\mathbf{v}_{ground}$ 为地速（相对于地面）。

### 2.2 风速剖面 (Wind Profile)

实际大气中风速随高度变化，使用**对数律**或**幂律**：

**对数律：**

$$
V_w(z) = V_{w,ref} \frac{\ln(z/z_0)}{\ln(z_{ref}/z_0)}
$$

**幂律：**

$$
V_w(z) = V_{w,ref} \left( \frac{z}{z_{ref}} \right)^\alpha
$$

其中 $z_0$ 为地表粗糙度，$\alpha$ 为幂律指数（典型值 0.1–0.4）。

| 地表类型 | 粗糙度 $z_0$ (m) | 幂律指数 $\alpha$ |
|---------|-----------------|------------------|
| 水面 | 0.0002 | 0.10 |
| 草地 | 0.03 | 0.16 |
| 郊区 | 0.5 | 0.28 |
| 城市 | 1.0 | 0.40 |

## 3. Dryden 大气湍流模型

### 3.1 概述

Dryden 模型是军用规范 MIL-HDBK-1797 和 MIL-F-8785C 中定义的标准湍流模型，使用**有理传递函数**来生成湍流风速序列。

### 3.2 传递函数

**水平湍流速度 ($u_g$)：**

$$
\Phi_u(\omega) = \sigma_u^2 \frac{2 L_u}{\pi V} \frac{1}{1 + (L_u \omega / V)^2}
$$

对应的传递函数：

$$
H_u(s) = \sigma_u \sqrt{\frac{2 L_u}{\pi V}} \frac{1}{1 + (L_u / V) s}
$$

**垂直湍流速度 ($w_g$)：**

$$
H_w(s) = \sigma_w \sqrt{\frac{L_w}{\pi V}} \frac{1 + (\sqrt{3} L_w / V) s}{(1 + (L_w / V) s)^2}
$$

**横向湍流速度 ($v_g$)：**

$$
H_v(s) = \sigma_v \sqrt{\frac{2 L_v}{\pi V}} \frac{1}{1 + (L_v / V) s}
$$

### 3.3 参数表

**湍流强度** (MIL-F-8785C)：

| 湍流等级 | $\sigma_u$ (m/s) | 说明 |
|---------|------------------|------|
| 轻度 (Light) | 1.5 | 晴天，高空 |
| 中度 (Moderate) | 3.0 | 一般天气 |
| 重度 (Severe) | 6.0 | 强对流天气 |

**湍流长度尺度** (低空 $h < 300$m)：

$$
L_u = \frac{h}{(0.177 + 0.000823 h)^{1.2}}
$$

$$
L_w = h
$$

$$
\sigma_w = \sigma_u
$$

**高空 ($h > 300$m)：**

$$
L_u = L_v = 530 \text{ m}
$$

$$
L_w = 530 \text{ m}
$$

$$
\sigma_u = \sigma_v = \sigma_w = 1.5 \text{ m/s (轻度)}
$$

```matlab
% Dryden 湍流模型参数
function params = dryden_params(h, turbulence_level)
% h: 飞行高度 (m)
% turbulence_level: 'light', 'moderate', 'severe'

    switch turbulence_level
        case 'light'
            sigma_u = 1.5;
        case 'moderate'
            sigma_u = 3.0;
        case 'severe'
            sigma_u = 6.0;
    end

    % 长度尺度
    if h < 300
        Lu = h / (0.177 + 0.000823*h)^1.2;
        Lw = h;
    else
        Lu = 530;
        Lw = 530;
    end
    Lv = Lu;

    params.sigma_u = sigma_u;
    params.sigma_v = sigma_u;
    params.sigma_w = sigma_u;
    params.Lu = Lu;
    params.Lv = Lv;
    params.Lw = Lw;
end

% Simulink Transfer Fcn 模块参数
% H_u(s) = sigma_u * sqrt(2*Lu/(pi*V)) / (1 + (Lu/V)*s)
% num_u = sigma_u * sqrt(2*Lu/(pi*V))
% den_u = [Lu/V, 1]
```

### 3.5 完整 MATLAB 实现

以下给出一个可直接调用的 Dryden 湍流模型函数，基于 MIL-F-8785C 规范，采用离散化递推滤波器实现，无需 Simulink 依赖。

```matlab
function [u_w, v_w, w_w] = dryden_wind_model(V, h, dt, t_end)
% Dryden Wind Turbulence Model (MIL-F-8785C)
% V: airspeed (m/s), h: altitude (m), dt: time step (s)
    
    % Turbulence intensities (moderate turbulence)
    sigma_u = 0.1 * V;  % longitudinal
    sigma_v = 0.1 * V;  % lateral
    sigma_w = 0.1 * V;  % vertical
    
    % Length scales (MIL-F-8785C)
    L_u = h / (0.177 + 0.000823*h)^1.2;  % longitudinal
    L_v = L_u;  % lateral
    L_w = h;    % vertical
    
    % Transfer function parameters (shaping filters)
    % H_u(s) = sigma_u * sqrt(2*L_u/(pi*V)) / (1 + L_u/V * s)
    % H_v(s) = sigma_v * sqrt(L_v/(pi*V)) * (1 + sqrt(3)*L_v/V*s) / (1 + L_v/V*s)^2
    % H_w(s) = sigma_w * sqrt(L_w/(pi*V)) * (1 + sqrt(3)*L_w/V*s) / (1 + L_w/V*s)^2
    
    n = round(t_end / dt);
    white = randn(3, n);
    
    % Filter coefficients (discrete approximation)
    a_u = exp(-V*dt/L_u);
    a_v = exp(-V*dt/L_v);
    a_w = exp(-V*dt/L_w);
    
    K_u = sigma_u * sqrt(2*V/(pi*L_u)) * sqrt(1 - a_u^2);
    K_v = sigma_v * sqrt(V/(pi*L_v)) * sqrt(1 - a_v^2);
    K_w = sigma_w * sqrt(V/(pi*L_w)) * sqrt(1 - a_w^2);
    
    u_w = zeros(1,n); v_w = zeros(1,n); w_w = zeros(1,n);
    for i = 2:n
        u_w(i) = a_u * u_w(i-1) + K_u * white(1,i);
        v_w(i) = a_v * v_w(i-1) + K_v * white(2,i);
        w_w(i) = a_w * w_w(i-1) + K_w * white(3,i);
    end
end
```

### 3.6 6DOF 仿真集成示例

以下示例展示如何将 Dryden 湍流风速叠加到 6DOF 模型中，模拟风扰下的飞行器响应。

```matlab
%% Dryden 湍流风扰 + 6DOF 仿真集成示例
% 演示如何将 dryden_wind_model 的输出注入 6DOF 动力学模型

clear; clc; close all;

%% 1. 仿真参数
dt      = 0.01;       % 时间步长 (s)
t_end   = 60;         % 仿真时长 (s)
t       = 0:dt:t_end;
n       = length(t);

%% 2. 飞行器初始状态
% 状态向量: [x, y, z, u, v, w, phi, theta, psi, p, q, r]
state = zeros(12, 1);
state(3)  = -100;     % 初始高度 100m (NED, z 向下为正)
state(4)  = 15;       % 初始空速 15 m/s

%% 3. 生成 Dryden 湍流风速序列
V_air   = 15;         % 巡航空速 (m/s)
h_agl   = 100;        % 离地高度 (m)

[u_w, v_w, w_w] = dryden_wind_model(V_air, h_agl, dt, t_end);

%% 4. 添加稳态风分量
V_const     = 5;                    % 稳态风速 (m/s)
chi_wind    = deg2rad(45);          % 风向角 (从东北来)
u_const     = V_const * cos(chi_wind);
v_const     = V_const * sin(chi_wind);

% 总风速 = 稳态风 + 湍流
u_wind_total = u_const + u_w;
v_wind_total = v_const + v_w;
w_wind_total = w_w;

%% 5. 简化 6DOF 仿真循环（悬停保持模式）
Kp_pos = 0.8;   % 位置环比例增益
Kp_vel = 1.5;   % 速度环比例增益
Kp_att = 3.0;   % 姿态环比例增益

pos_log  = zeros(3, n);
vel_log  = zeros(3, n);
wind_log = zeros(3, n);

for i = 1:n
    % 当前风速
    wind = [u_wind_total(i); v_wind_total(i); w_wind_total(i)];
    wind_log(:, i) = wind;
    
    % 空速 = 地速 - 风速
    v_air = state(4:6) - wind;
    
    % 简化气动力（线性模型）
    Cd = 0.3; S = 0.1; rho = 1.225;
    F_drag = -0.5 * rho * norm(v_air) * Cd * S * v_air;
    
    % 位置保持控制器（悬停在 [0, 0, -100]）
    pos_des = [0; 0; -100];
    vel_des = [0; 0; 0];
    
    pos_err = pos_des - state(1:3);
    vel_cmd = Kp_pos * pos_err;
    vel_err = vel_cmd - state(4:6);
    accel_cmd = Kp_vel * vel_err;
    
    % 姿态指令（简化：直接映射加速度到姿态）
    theta_cmd = -accel_cmd(1) / 9.81;   % 俯仰 -> 纵向加速度
    phi_cmd   =  accel_cmd(2) / 9.81;   % 滚转 -> 横向加速度
    
    % 姿态控制
    att_err = [phi_cmd; theta_cmd; 0] - state(7:9);
    omega_cmd = Kp_att * att_err;
    
    % 力矩计算
    J = diag([0.01, 0.01, 0.02]);  % 转动惯量
    tau = J * omega_cmd;
    
    % 动力学积分（前向 Euler）
    % 合力 = 重力 + 推力 + 气动阻力
    m = 1.5;  % 质量 (kg)
    g = 9.81;
    R = euler2rotm(state(7:9));  % 欧拉角 -> 旋转矩阵
    F_thrust = R' * [0; 0; -m*g - accel_cmd(3)*m];  % 推力
    F_total = [0; 0; m*g] + F_thrust + F_drag;      % NED 重力 + 推力 + 阻力
    
    % 状态更新
    state(1:3) = state(1:3) + state(4:6) * dt;       % 位置
    state(4:6) = state(4:6) + (F_total / m) * dt;    % 速度
    state(7:9) = state(7:9) + state(10:12) * dt;     % 姿态
    state(10:12) = state(10:12) + (J \ tau) * dt;     % 角速度
    
    % 记录
    pos_log(:, i) = state(1:3);
    vel_log(:, i) = state(4:6);
end

%% 6. 绘制结果
figure('Name', 'Dryden Wind Turbulence + 6DOF Response', 'Position', [100 100 1200 800]);

% 风速时域信号
subplot(3,2,1);
plot(t, u_wind_total, 'b', t, v_wind_total, 'r', t, w_wind_total, 'g');
xlabel('时间 (s)'); ylabel('风速 (m/s)');
legend('u (纵向)', 'v (横向)', 'w (垂直)');
title('Dryden 湍流风速 (含稳态风)'); grid on;

% 位置响应
subplot(3,2,2);
plot(t, pos_log(1,:), 'b', t, pos_log(2,:), 'r', t, -pos_log(3,:), 'g');
xlabel('时间 (s)'); ylabel('位置 (m)');
legend('x', 'y', 'z (高度)');
title('飞行器位置响应'); grid on;

% 位置偏差
subplot(3,2,3);
pos_err_log = pos_log - [0; 0; -100] * ones(1,n);
plot(t, pos_err_log(1,:), 'b', t, pos_err_log(2,:), 'r', t, pos_err_log(3,:), 'g');
xlabel('时间 (s)'); ylabel('位置偏差 (m)');
legend('\Deltax', '\Deltay', '\Deltaz');
title('位置保持误差'); grid on;

% 速度响应
subplot(3,2,4);
plot(t, vel_log(1,:), 'b', t, vel_log(2,:), 'r', t, vel_log(3,:), 'g');
xlabel('时间 (s)'); ylabel('速度 (m/s)');
legend('u', 'v', 'w');
title('飞行器速度响应'); grid on;

%% 7. 功率谱密度分析
subplot(3,2,[5 6]);
Fs = 1/dt;
NFFT = 2^nextpow2(n);

% 纵向湍流 PSD
[pxx_u, f_u] = pwelch(u_w, hann(1024), 512, NFFT, Fs);
% 横向湍流 PSD
[pxx_v, f_v] = pwelch(v_w, hann(1024), 512, NFFT, Fs);
% 垂直湍流 PSD
[pxx_w, f_w] = pwelch(w_w, hann(1024), 512, NFFT, Fs);

loglog(f_u, pxx_u, 'b', 'LineWidth', 1.2); hold on;
loglog(f_v, pxx_v, 'r', 'LineWidth', 1.2);
loglog(f_w, pxx_w, 'g', 'LineWidth', 1.2);

% 叠加理论 Dryden PSD (纵向) 用于验证
Lu = h_agl / (0.177 + 0.000823*h_agl)^1.2;
sigma_u_theory = 0.1 * V_air;
omega = 2*pi*f_u;
Phi_theory = sigma_u_theory^2 * (2*Lu/(pi*V_air)) ./ (1 + (Lu*omega/V_air).^2);
Phi_theory = Phi_theory / (2*pi);  % 转换为单边 PSD (Hz)
loglog(f_u, Phi_theory, 'k--', 'LineWidth', 1.5);

xlabel('频率 (Hz)');
ylabel('功率谱密度 (m^2/s^2/Hz)');
legend('u (纵向)', 'v (横向)', 'w (垂直)', 'Dryden 理论 \Phi_u');
title('Dryden 湍流功率谱密度 (PSD)');
grid on; xlim([0.001, Fs/2]);

sgtitle('Dryden 风扰模型仿真验证', 'FontSize', 14);

%% 辅助函数
function R = euler2rotm(euler)
% 欧拉角 [phi, theta, psi] -> 旋转矩阵 (ZYX 顺序)
    phi   = euler(1);
    theta = euler(2);
    psi   = euler(3);
    
    Rx = [1 0 0; 0 cos(phi) -sin(phi); 0 sin(phi) cos(phi)];
    Ry = [cos(theta) 0 sin(theta); 0 1 0; -sin(theta) 0 cos(theta)];
    Rz = [cos(psi) -sin(psi) 0; sin(psi) cos(psi) 0; 0 0 1];
    
    R = Rz * Ry * Rx;
end
```

### 3.7 结果分析说明

上述仿真产生三组关键输出：

| 输出 | 用途 |
|------|------|
| 风速时域信号 | 验证湍流强度和时间相关性是否符合 MIL-F-8785C |
| 位置/速度响应 | 评估控制器在湍流风扰下的保持精度 |
| 功率谱密度 (PSD) | 验证生成的湍流信号是否匹配 Dryden 理论谱 |

**PSD 验证要点：** 实际仿真的 PSD 曲线应与理论曲线在低频段吻合，高频段由于离散化效应会有一定偏差。如果偏差过大，需要减小时间步长 `dt` 或增大湍流长度尺度的离散化精度。

## 4. Von Karman 大气湍流模型

### 4.1 与 Dryden 的区别

| 特性 | Dryden | Von Karman |
|------|--------|------------|
| 功率谱密度形式 | 有理函数 | 非有理函数 |
| 传递函数 | 可直接实现 | 需要近似 |
| 高频特性 | $\omega^{-2}$ | $\omega^{-5/3}$ |
| 物理精度 | 较低 | 更高 |
| 实现难度 | 简单 | 较复杂 |
| 标准 | MIL-F-8785C | MIL-HDBK-1797 |

### 4.2 Von Karman 功率谱密度

$$
\Phi_u(\omega) = \sigma_u^2 \frac{2 L_u}{\pi V} \frac{1}{[1 + (1.339 L_u \omega / V)^2]^{5/6}}
$$

由于非有理形式，通常使用高阶有理函数近似来实现 Simulink 模型。

### 4.3 选择建议

- **仿真验证/快速原型**：使用 Dryden（简单、计算快）
- **飞行品质评估**：使用 Von Karman（更精确、符合新标准）
- **控制系统设计**：两者皆可，差异通常不大

## 5. 风阵风模型

### 5.1 阶跃阵风 (Step Gust)

最简单的阵风模型，风速在瞬间从 0 跳变到 $V_{gust}$：

$$
V_w(t) = V_{gust} \cdot u(t - t_0)
$$

其中 $u(t)$ 为单位阶跃函数。

### 5.2 离散阵风 (Discrete Gust, 1-cosine)

MIL-F-8785C 定义的 1-cosine 阵风：

$$
V_g(x) = \begin{cases} \frac{V_{g,max}}{2} \left(1 - \cos\frac{\pi x}{d_m}\right) & 0 \leq x \leq 2d_m \\ 0 & \text{otherwise} \end{cases}
$$

其中 $d_m$ 为阵风梯度距离（典型值 30–120 m）。

```
  V_g
   ▲
   │       ┌─────────┐
   │      ╱│         │╲
   │    ╱  │         │  ╲
   │  ╱    │         │    ╲
   │╱      │         │      ╲
   └───────┼─────────┼──────────► x (距离)
   0      d_m       2d_m
```

### 5.3 连续阵风 (Continuous Gust)

连续湍流阵风通过功率谱密度生成，实际上就是 Dryden 或 Von Karman 模型。

### 5.4 Simulink 实现

```matlab
% 1-cosine 阵风 Simulink 实现
function V_gust = discrete_gust_1cosine(x, V_max, d_m)
    if x >= 0 && x <= 2*d_m
        V_gust = V_max/2 * (1 - cos(pi * x / d_m));
    else
        V_gust = 0;
    end
end

% 典型参数
gust.V_max = 15;    % m/s 最大阵风速度
gust.d_m = 50;      % m 阵风梯度距离
gust.direction = [1; 0; 0];  % 沿 x 方向
```

### 7.2 MATLAB 数据导入

```matlab
% 从 ERA5 数据导入风场
function wind_field = load_era5_wind(filename, lat, lon, time)
    ncid = netcdf.open(filename, 'NOWRITE');

    % 读取 u, v 分量
    u = netcdf.getVar(ncid, u_varid);  % 东西分量
    v = netcdf.getVar(ncid, v_varid);  % 南北分量

    % 插值到指定位置和时间
    u_interp = interp3(lon_grid, lat_grid, time_grid, u, lon, lat, time);
    v_interp = interp3(lon_grid, lat_grid, time_grid, v, lon, lat, time);

    wind_field.u = u_interp;
    wind_field.v = v_interp;
    wind_field.speed = sqrt(u_interp^2 + v_interp^2);
    wind_field.direction = atan2d(-u_interp, -v_interp);
end
```

## 9. 悬挂载荷动力学

### 9.1 带悬挂载荷的风扰影响

当无人机携带悬挂载荷（如救援物资投放）时，风扰会激发载荷摆动：

```
        ┌─────┐
        │ UAV │
        └──┬──┘
           │ 绳长 L
           │
           │  θ (摆角)
           │
           ○ 载荷
           ←── 风力
```

**载荷摆动方程（简化单摆模型）：**

$$
\ddot{\theta} + \frac{c}{mL}\dot{\theta} + \frac{g}{L}\sin\theta = -\frac{\ddot{x}_{UAV}}{L}\cos\theta + \frac{F_{wind}}{mL}
$$

### 9.2 Simulink 悬挂载荷模型

```matlab
% 悬挂载荷参数
payload.mass = 2;           % kg
payload.cable_length = 3;   % m
payload.cable_damping = 0.1; % Ns/m
payload.drag_area = 0.05;   % m^2 (载荷迎风面积)
payload.Cd = 0.8;           % 载荷阻力系数

% 风对载荷的力
F_wind_payload = 0.5 * rho * V_wind^2 * payload.Cd * payload.drag_area;
```

## 思考题

1. **Dryden 模型和 Von Karman 模型在功率谱密度上有什么本质区别？为什么 Von Karman 被认为更精确？**

<details><summary>参考答案</summary>
Dryden 模型的功率谱密度在高频段以 $\omega^{-2}$ 衰减，而 Von Karman 模型以 $\omega^{-5/3}$ 衰减。$\omega^{-5/3}$ 衰减率对应 Kolmogorov 湍流理论中的惯性子区间，是真实大气湍流的物理规律。Dryden 模型的 $\omega^{-2}$ 衰减是为了获得有理传递函数（便于 Simulink 实现）而做的数学近似。Von Karman 模型更精确，但需要高阶有理近似来实现。对于飞行品质评估（关注湍流的高频影响），Von Karman 更可靠；对于控制器设计（关注低频特性），两者差异不大。
</details>

2. **在 Simulink 中实现 Dryden 模型时，传递函数的输入应该是什么信号？为什么？**

<details><summary>参考答案</summary>
Dryden 传递函数的输入应该是**白噪声**（Band-Limited White Noise 模块）。这是因为 Dryden 模型的功率谱密度已经编码了湍流的统计特性（强度、尺度），白噪声具有平坦的功率谱密度（在所有频率上能量相同），通过传递函数滤波后，输出信号的功率谱密度恰好匹配目标湍流谱。白噪声模块的采样时间应设为远小于湍流最小时间常数（$T_s << L_w/V$），通常取 0.01s 量级。噪声强度设为 1 即可，因为幅度缩放由传递函数的增益完成。
</details>

3. **如何确定仿真中应该使用什么湍流等级？有哪些参考标准？**

<details><summary>参考答案</summary>
湍流等级选择依据：(1) **MIL-F-8785C** 标准定义了轻度（晴天高空）、中度（一般天气）、重度（强对流）三个等级，是最常用的参考；(2) **飞行任务需求**——民用无人机通常按中度湍流设计，军用需考虑重度；(3) **实际气象条件**——根据任务区域的气象统计选择，如沿海地区风大，山区湍流强；(4) **认证要求**——适航标准（如 EASA SC-VTOL）对不同飞行阶段有明确的湍流要求；(5) **保守原则**——设计验证时至少测试比预期高一级的湍流等级，留出安全裕度。
</details>

4. **风对悬挂载荷的影响与对无人机本身的影响有何不同？如何在控制器中补偿载荷摆动？**

<details><summary>参考答案</summary>
风对载荷的影响更严重，因为：(1) 载荷通常阻力系数大（非流线型）；(2) 绳索为柔性约束，摆动阻尼很小；(3) 载荷摆动产生的惯性力会反作用于无人机，影响姿态。补偿方法：(1) **前馈补偿**——通过检测风速估算载荷摆动趋势，提前修正无人机位置；(2) **状态估计**——使用机载 IMU 和载荷 IMU（如有）估计摆角，反馈到控制器；(3) **输入整形**——在轨迹规划中加入防摆动约束，避免激励摆动模态；(4) **主动阻尼**——通过无人机的加速/减速运动主动抑制载荷摆动（类似于起重机的防摇控制）。
</details>

5. **如果要在 Simulink 中集成真实气象数据（如 ERA5），应该注意哪些工程问题？**

<details><summary>参考答案</summary>
主要工程问题：(1) **数据分辨率**——ERA5 时间分辨率为 1 小时，空间分辨率 0.25°（约 25km），对于单次飞行仿真需要时间/空间插值；(2) **坐标系转换**——气象数据使用地理坐标（经纬度），仿真需要 NED 或 ENU 坐标系，需要投影转换；(3) **数据量**——一次飞行可能需要多个高度层的三维风场，数据量大，建议预处理为查找表；(4) **实时性**——NetCDF 读取有一定延迟，建议预加载到 MATLAB workspace；(5) **缺失值处理**——气象数据可能有缺失，需要插值或使用备用数据源；(6) **单位一致性**——ERA5 风速单位为 m/s，温度为 K，注意与仿真单位匹配。
</details>
