## 1. 电机-电调-旋翼系统概述

无人机的动力链路从电池到旋翼推力，经过多个环节。每个环节都会引入动态特性和能量损耗。

```
 ┌──────┐    ┌─────┐    ┌──────┐    ┌─────┐    ┌──────┐
 │      │    │     │    │      │    │     │    │      │
 │ 电池 │───→│ ESC │───→│ 电机 │───→│ 螺旋桨│───→│ 推力 │
 │      │    │     │    │      │    │     │    │      │
 └──────┘    └─────┘    └──────┘    └─────┘    └──────┘
   电压       PWM→       电能→       旋转→      气动
   内阻       电压       机械能       推力       力矩
```

### 1.1 各环节特性对比

| 环节 | 输入 | 输出 | 动态特性 | 时间常数 |
|------|------|------|---------|---------|
| 电池 | 负载电流 | 电压 | 准静态（缓慢下降） | 秒级 |
| ESC | PWM 信号 | 三相电压 | 近似无延迟 | < 1ms |
| 电机 | 电压 | 转速 | 一阶惯性 | 10-50ms |
| 螺旋桨 | 转速 | 推力/扭矩 | 近似无延迟 | < 1ms |

> **关键动态**：电机的机电时间常数（10-50ms）是整个动力系统中最主要的动态环节，决定了推力响应速度。

## 2. BLDC 电机基础

### 2.1 无刷直流电机（BLDC）工作原理

四旋翼使用外转子无刷直流电机（Outrunner BLDC）：

```
    外转子 BLDC 结构（截面图）：

         ┌─────────────────────┐
         │  永磁体（N/S交替）   │ ← 外转子（旋转）
         │  ┌─────────────┐    │
         │  │   定子线圈   │    │ ← 内定子（固定）
         │  │  (三相绕组)  │    │
         │  └─────────────┘    │
         │                     │
         └─────────────────────┘

    三相绕组：A, B, C（星形连接）
    电子换向：ESC 按顺序切换三相通电
```

### 2.2 BLDC 等效电路模型

```
        ┌──────┐     ┌──────┐     ┌──────┐
   V ──┤  R   ├──┤  L   ├──┤  Ke·ω│──→ 反电动势
        │(电阻)│     │(电感)│     │(反EMF)│
        └──────┘     └──────┘     └──────┘
             │                         │
             └────── I (电流) ─────────┘
```

电机电压方程：

$$
V = R \cdot I + L \cdot \frac{dI}{dt} + K_e \cdot \omega
$$

| 参数 | 符号 | 含义 | 典型值 |
|------|------|------|--------|
| 端电压 | V | 电机输入电压 | 7.4-22.2V |
| 绕组电阻 | R | 相间电阻 | 0.1-1.0 Ω |
| 绕组电感 | L | 相间电感 | 10-100 μH |
| 反电动势常数 | Ke | V/(rad/s) | 0.01-0.1 V·s/rad |
| 转矩常数 | Kt | N·m/A | = Ke（SI 单位制） |
| 转速 | ω | 输出角速度 | 0-12000 rad/s |

> **注意**：在 SI 单位制中，Kt = Ke。这是因为电机是可逆的：电动机和发电机的常数相同。

### 2.3 电机机械方程

$$
J_m \cdot \frac{d\omega}{dt} = K_t \cdot I - Q_{load} - B \cdot \omega
$$

| 项 | 含义 |
|----|------|
| J_m·dω/dt | 转子惯性力矩 |
| Kt·I | 电磁转矩 |
| Q_load | 负载扭矩（螺旋桨阻力） |
| B·ω | 摩擦力矩（通常很小） |

## 3. 一阶电机模型

### 3.1 简化推导

当电感 L 很小（L/R 时间常数 << 机械时间常数）时，忽略电感动态：

$$
I = \frac{V - K_e \cdot \omega}{R}
$$

代入机械方程：

$$
J_m \cdot \dot{\omega} = K_t \cdot \frac{V - K_e \omega}{R} - Q_{load} - B\omega
$$

整理：

$$
J_m \cdot \dot{\omega} + \left(\frac{K_t K_e}{R} + B\right) \omega = \frac{K_t}{R} V - Q_{load}
$$

### 3.2 传递函数

忽略负载扭矩（线性化分析），对上式进行拉普拉斯变换：

$$
\frac{\Omega(s)}{V(s)} = \frac{K_t / (R \cdot J_m)}{s + \frac{K_t K_e + R \cdot B}{R \cdot J_m}}
$$

简化为一阶惯性环节：

$$
\frac{\Omega(s)}{V(s)} = \frac{K_{motor}}{\tau_m s + 1}
$$

| 参数 | 表达式 | 物理含义 |
|------|--------|---------|
| 增益 K_motor | K_t / (K_t·K_e + R·B) | 稳态转速/输入电压 |
| 时间常数 τ_m | R·J_m / (K_t·K_e + R·B) | 响应速度 |

### 3.3 典型参数值

| 参数 | 小型电机 (2204) | 中型电机 (2212) | 大型电机 (4010) |
|------|----------------|----------------|----------------|
| R (Ω) | 0.2 | 0.15 | 0.08 |
| L (μH) | 15 | 25 | 50 |
| Ke (V·s/rad) | 0.015 | 0.025 | 0.05 |
| Jm (kg·m²) | 2×10⁻⁶ | 5×10⁻⁶ | 2×10⁻⁵ |
| τ_m (ms) | 15 | 25 | 40 |
| K_motor (rad/s/V) | 400 | 280 | 150 |

## 4. 二阶电机模型（含 ESC 动态）

### 4.1 ESC 建模

ESC（Electronic Speed Controller）接收 PWM 信号，输出三相交流驱动电机：

```
    输入：PWM 信号（1000-2000 μs）
          │
          ▼
    ┌───────────┐
    │    ESC     │
    │            │
    │ PWM → 占空比│──→ 三相电压 ──→ 电机
    │ → 电压映射  │
    └───────────┘

    DShot 协议：数字信号，直接传输油门值
    PWM 协议：脉宽 1000-2000μs 映射到 0-100% 油门
```

### 4.2 ESC 传递函数

ESC 的动态可以用一个带有延迟的一阶系统近似：

$$
\frac{V_{out}(s)}{PWM(s)} = \frac{K_{ESC}}{\tau_{ESC} s + 1} \cdot e^{-t_d s}
$$

| 参数 | 含义 | 典型值 |
|------|------|--------|
| K_esc | PWM 到电压增益 | V_batt / 1000 (V/μs) |
| τ_esc | ESC 时间常数 | 1-5 ms |
| t_d | 死区/传输延迟 | 0.1-1 ms |

### 4.3 完整二阶模型

将 ESC 和电机串联：

$$
\frac{\Omega(s)}{PWM(s)} = \frac{K_{ESC} \cdot K_{motor}}{(\tau_{ESC} s + 1)(\tau_m s + 1)}
$$

当 τ_esc << τ_m 时，可简化为一阶模型（ESC 忽略）。对于大多数工程应用，一阶模型已足够。

## 5. 螺旋桨负载模型

### 5.1 负载扭矩

螺旋桨旋转时产生空气阻力扭矩：

$$
Q_{prop} = k_Q \cdot \omega^2
$$

这是一个 **非线性** 负载，转速越高，负载越大。

### 5.2 负载对电机响应的影响

在电机方程中代入负载：

$$
J_m \cdot \dot{\omega} = K_t \cdot I - k_Q \cdot \omega^2 - B \cdot \omega
$$

与无负载情况相比：
- **增益降低**：稳态转速下降
- **时间常数改变**：非线性负载使响应曲线不对称（加速慢于减速）
- **非线性特性**：传递函数不再是简单的线性系统

### 5.3 稳态工作点

在稳态时（dω/dt = 0）：

$$
\frac{K_t}{R}(V - K_e \omega_{ss}) = k_Q \omega_{ss}^2 + B\omega_{ss}
$$

这是一个关于 ω_ss 的二次方程，可以通过数值方法求解。

## 6. 电池模型

### 6.1 等效电路

```
    ┌────────────┐     ┌──────────┐
    │  开路电压   │     │  内阻    │
    │  V_oc(SOC) ├──┤──┤  R_int  ├──→ V_terminal
    └────────────┘     └──────────┘
         │                   │
         └──── SOC (荷电状态) ────┘
```

$$
V_{terminal} = V_{oc}(SOC) - I \cdot R_{int}(SOC)
$$

### 6.2 锂电池放电特性

| 参数 | 含义 | LiPo 3S (11.1V) 典型值 |
|------|------|----------------------|
| 满充电压 | 每节 4.2V | 12.6V |
| 标称电压 | 每节 3.7V | 11.1V |
| 截止电压 | 每节 3.0V | 9.0V |
| 内阻 R_int | 总内阻 | 20-100 mΩ |
| 容量 C | 额定容量 | 1000-5000 mAh |

### 6.3 电压降模型

电池电压随放电电流和 SOC 变化：

$$
V_{batt}(SOC, I) = N_{cell} \cdot [V_{cell}(SOC) - I \cdot R_{cell}]
$$

典型的电压-SOC 曲线：

```
    电压 (V/cell)
    4.2 │━━━━━┓
        │      ┃
    4.0 │      ┗━━━━━━━━━━━━━┓
        │                    ┃
    3.7 │                    ┗━━━━━━━━┓
        │                             ┃
    3.3 │                             ┗━━┓
        │                                ┃
    3.0 │                                ┗━━ ← 截止
        └──────────────────────────────────── SOC
        100%  80%  60%  40%  20%   0%
```

> **工程影响**：满电和低电量时，同样的油门指令产生不同的推力。高性能飞控需要补偿电池电压变化。

## 7. 电机故障建模

### 7.1 故障类型

| 故障类型 | 表现 | 建模方法 | 发生概率 |
|---------|------|---------|---------|
| 电机失效 | 转速归零 | T = 0, Q = 0 | 中 |
| 效率下降 | 响应变慢 | 增大 τ_m | 高 |
| 间歇断电 | 转速波动 | 噪声叠加 | 低 |
| 轴承损坏 | 阻力增大 | 增大 B | 中 |
| 桨叶断裂 | kT/kQ 改变 | 修改旋翼参数 | 低 |

### 7.2 失效建模

电机完全失效的 Simulink 实现：

```
    正常路径：  ω_cmd ──→ [Motor Model] ──→ ω_actual
    
    故障注入：  fault_flag ──→ ┌─────────────┐
                               │ Switch      │
                   ω_cmd ──→  │ if flag=0:  │──→ ω_actual
                   0     ──→  │   正常       │
                               │ else:       │
                               │   输出 0    │
                               └─────────────┘
```

### 7.3 故障检测

通过监测电机转速与指令的偏差来检测故障：

$$
e_{motor} = \omega_{cmd} - \omega_{actual}
$$

如果 $|e_{motor}| > threshold$ 持续一段时间，则判断电机故障。

## 8. Simulink 实现

### 8.1 电机子系统架构

```
┌────────────────────────────────────────────────────────┐
│              Motor + ESC + Propeller Subsystem          │
│                                                        │
│  油门指令 ──→ ┌────────┐    ┌────────┐    ┌────────┐  │
│  (0-1)       │ ESC    │───→│ Motor  │───→│Propeller│  │
│              │ Model  │    │ Model  │    │ Model   │  │
│              │PWM→V   │    │V→ω     │    │ω→T,Q   │  │
│              └────────┘    └────────┘    └────────┘  │
│                                  │            │       │
│                                  ↓            ↓       │
│                              ω (rad/s)    T, Q 输出   │
│                                                        │
│  ┌──────────────────────────────────────────────────┐  │
│  │ Battery Model: V_batt(SOC, I_total)             │  │
│  └──────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────┘
```

### 8.2 MATLAB Function 示例代码

```matlab
function [T, Q, omega] = MotorPropModel(throttle, V_batt, params)
% Motor-Propeller combined model
% Inputs:
%   throttle: 0-1 normalized throttle command
%   V_batt: battery voltage (V)
%   params: struct with motor/prop parameters
% Outputs:
%   T: thrust (N)
%   Q: torque (N*m)
%   omega: motor speed (rad/s)

    % ESC: throttle to voltage
    V_motor = throttle * V_batt;
    
    % Motor: first-order dynamics (simplified steady-state)
    % omega = V_motor * K_motor (no dynamics for simplicity)
    omega_ss = V_motor * params.K_motor;
    
    % Propeller: thrust and torque
    T = params.kT * omega_ss^2;
    Q = params.kQ * omega_ss^2;
    
    omega = omega_ss;
end
```

### 8.3 含动态的 Integrator 版本

```matlab
function [omega_dot] = MotorDynamics(omega, V_motor, Q_prop, params)
% Motor dynamics: J * dω/dt = Kt*I - Q_load - B*ω
% Simplified: τ * dω/dt = omega_cmd - omega

    omega_cmd = V_motor * params.K_motor;
    tau = params.tau_m;
    
    omega_dot = (omega_cmd - omega) / tau;
end
```

## 思考题

**1. 为什么在大多数四旋翼仿真中，可以将 ESC 和电机简化为一阶系统，而不需要建模电感 L？**

<details>
<summary>参考答案</summary>

电感的时间常数 τ_L = L/R。对于典型 BLDC 电机：
- L ≈ 15-50 μH
- R ≈ 0.1-1.0 Ω
- τ_L = L/R ≈ 15-500 μs = 0.015-0.5 ms

而机械时间常数 τ_m ≈ 10-50 ms。

比值 τ_L/τ_m ≈ 0.001-0.05，即电感动态比机械动态快 20-100 倍。

在控制理论中，当一个子系统的时间常数远小于主系统时，可以将其简化为静态增益（零阶近似）或忽略。这就是为什么：
- ESC 通常建模为增益（PWM → 电压）
- 电机电路忽略电感，只保留电阻

只有在需要精确建模电流纹波或研究 FOC（磁场定向控制）时，才需要考虑电感动态。
</details>

**2. 电池电压下降对飞行器性能有什么影响？如何在飞控中补偿？**

<details>
<summary>参考答案</summary>

电池电压下降的影响：

1. **推力下降**：同样油门值，电压降低 → 电机转速降低 → 推力减小
2. **响应变慢**：电压降低 → K_motor 降低 → 时间常数增大 → 响应变慢
3. **最大推力受限**：低电压时即使满油门也无法达到设计推力

补偿方法：

1. **电压前馈补偿**：
   - 测量电池电压 V_batt
   - 油门补偿：throttle_comp = throttle_nominal * V_nominal / V_batt
   - PX4 中称为 "voltage compensation" 或 "BAT scaling"

2. **油门曲线校正**：
   - 根据电压-推力关系修正油门映射表
   - 低电压时自动提高基础油门

3. **自适应控制**：
   - 在线估计 kT 参数
   - 根据实际推力调整控制增益

4. **电池 SOC 估计**：
   - 使用库仑计数或电压法估计剩余电量
   - 在电量低时限制机动或自动返航

PX4 默认启用电压补偿，参数为 BAT_* 系列。
</details>

**3. 如果一个电机的 kT 比其他三个低 10%（如桨叶损坏），四旋翼会出现什么飞行动态？**

<details>
<summary>参考答案</summary>

假设旋翼 1 的 kT 降低了 10%，即 kT1 = 0.9·kT_nominal。

**稳态影响**：

1. **推力不平衡**：旋翼 1 产生比预期少 10% 的推力
2. **滚转力矩**：如果旋翼 1 在后方，产生低头力矩
3. **偏航力矩**：反扭矩不平衡，产生偏航趋势

**飞控响应**：

姿态控制器会检测到姿态偏差并增加其他旋翼的推力来补偿。但这会导致：
1. **总推力降低**：补偿推力损失需要增加其他旋翼，但受到最大转速限制
2. **效率下降**：非对称推力分配降低悬停效率
3. **机动能力受限**：可分配的控制余量减少

**数值估算**（简化）：
- 10% kT 损失 → 悬停时旋翼 1 需要增加约 5.4% 转速来补偿
- 如果电机已接近最大转速，可能无法完全补偿
- 偏航方向需要额外 2-3% 的对角转速差来平衡反扭矩

严重时（如 30%+ 损失），可能导致无法维持姿态，触发紧急着陆。
</details>

**4. 设计一个 Simulink 模型来验证一阶电机模型的阶跃响应，描述需要哪些模块和连接方式。**

<details>
<summary>参考答案</summary>

所需模块和连接：

1. **Step 模块**：模拟油门阶跃（0 → 0.7）
   - Step time: 0.1s
   - Initial value: 0
   - Final value: 0.7

2. **Gain 模块**：ESC 增益
   - Gain = V_batt / 1000 (假设 PWM 1000-2000 映射)

3. **Transfer Fcn 模块**：电机传递函数
   - Numerator: [K_motor]
   - Denominator: [tau_m, 1]
   - 参数：K_motor = 400, tau_m = 0.02

4. **Gain 模块**：推力系数
   - Gain = kT
   - 输出为推力 T

5. **Scope 模块**：显示转速和推力响应

连接：
Step → ESC Gain → Motor Transfer Fcn → Prop Gain (kT) → Scope
                    │
                    └──→ Scope（转速通道）

分析：
- 观察上升时间 t_r ≈ 2.2·τ_m
- 稳态值 = K_motor · V_batt · throttle · kT
- 验证 5% 调节时间 ≈ 3·τ_m
- 与实测数据对比，调整 K_motor 和 τ_m
</details>

**5. 为什么四旋翼通常使用外转子电机（Outrunner）而不是内转子电机？**

<details>
<summary>参考答案</summary>

外转子电机的优势：

1. **高扭矩/低转速**：外转子结构使永磁体半径更大，在相同磁通量下产生更大扭矩。四旋翼需要直接驱动螺旋桨（无减速齿轮），高扭矩低转速更合适。

2. **转动惯量大**：外转子的永磁体在外部，转动惯量大。虽然这降低了加速响应，但提供了更平稳的转速输出，减少转速波动。

3. **散热好**：旋转的外壳起到风扇作用，增强散热。高效率意味着更长续航。

4. **结构简单**：无需减速齿轮箱，减少重量、噪音和故障点。

5. **KV 值匹配**：外转子电机的 KV 值（rpm/V）通常在 500-1000 范围，配合 10-15 英寸螺旋桨效率最佳。

内转子电机主要用于：
- 需要极高转速的场景（如涵道风扇）
- 配合减速齿轮箱使用
- 微型无人机（受限于尺寸）

对于 95% 以上的多旋翼应用，外转子 BLDC 是标准选择。
</details>
