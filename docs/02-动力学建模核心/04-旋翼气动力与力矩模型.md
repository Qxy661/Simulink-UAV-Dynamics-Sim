## 1. 旋翼空气动力学基础

旋翼是四旋翼无人机的唯一动力来源。理解旋翼如何产生推力和力矩，是建立精确动力学模型的关键。

### 1.1 基本物理过程

```
    电机转动
       │
       ▼
   桨叶旋转 ──→ 切割空气 ──→ 产生升力（推力）
       │                        │
       ▼                        ▼
   空气阻力               反作用扭矩
   (消耗功率)             (使机架反向旋转)
```

### 1.2 关键参数

| 参数 | 符号 | 单位 | 说明 |
|------|------|------|------|
| 旋翼半径 | R | m | 桨叶长度 |
| 桨叶数 | N_b | - | 通常 2（双叶桨） |
| 旋转角速度 | Ω | rad/s | 电机转速 |
| 桨尖速度 | V_tip = Ω·R | m/s | 桨尖线速度 |
| 前进比 | μ = V/(Ω·R) | - | 前飞速度与桨尖速度之比 |
| 实度 | σ = N_b·c/(π·R) | - | 桨叶面积与桨盘面积之比 |
| 桨叶弦长 | c | m | 桨叶截面宽度 |

## 2. 动量理论（Actuator Disk Theory）

### 2.1 基本假设

动量理论将旋翼简化为一个 **致动盘（Actuator Disk）**：一个无限薄的圆盘，均匀地对流过它的空气施加力。

| 假设 | 说明 |
|------|------|
| 均匀滑流 | 流过桨盘的气流速度均匀 |
| 一维流动 | 只考虑轴向速度分量 |
| 不可压缩流 | 低速飞行（Ma < 0.3）有效 |
| 无旋转尾流 | 忽略旋翼对气流的旋转加速 |
| 无限远处来流 | 上游无扰动 |

### 2.2 推力公式

根据动量理论，旋翼推力：

$$
T = 2\rho A v_i \sqrt{V^2 + v_i^2}
$$

其中：
- ρ = 空气密度 (1.225 kg/m³)
- A = 桨盘面积 = πR²
- v_i = 诱导速度（桨盘处的下洗速度）
- V = 飞行速度（轴向分量）

**悬停时**（V = 0）：

$$
T = 2\rho A v_{i,h}^2
$$

$$
v_{i,h} = \sqrt{\frac{T}{2\rho A}}
$$

### 2.3 悬停诱导速度

| 参数 | 公式 | 500g 四旋翼典型值 |
|------|------|------------------|
| 单旋翼推力 | T = mg/4 | 1.226 N |
| 桨盘面积 | A = πR² | 0.00785 m² (R=0.05m) |
| 诱导速度 | v_i = √(T/2ρA) | 7.96 m/s |
| 悬停功率 | P = T·v_i | 9.76 W |

### 2.4 功率系数

| 系数 | 定义 | 物理含义 |
|------|------|---------|
| 推力系数 | C_T = T / (ρ·A·(ΩR)²) | 无量纲推力 |
| 功率系数 | C_P = P / (ρ·A·(ΩR)³) | 无量纲功率 |
| 扭矩系数 | C_Q = Q / (ρ·A·(ΩR)²·R) | 无量纲扭矩 |
| 品质因数 | FM = C_T^(3/2) / (√2·C_P) | 悬停效率（理想值=1） |

关系：C_P = C_Q·(2π)，因为 P = Q·Ω。

## 3. 叶素动量理论（BEMT）概述

### 3.1 基本思想

BEMT 将桨叶沿径向分成若干微段（叶素），对每个叶素独立分析气动力，再积分得到整片桨叶的力和力矩。

```
    桨叶径向分段示意：

    旋转中心            桨尖
        ●──────────────────────→ r
        │  │  │  │  │  │  │  │
        0  1  2  3  4  5  6  7  ← 叶素编号

    每个叶素：
    - 翼型截面（如 NACA 0012, Clark-Y）
    - 局部迎角 α
    - 升力系数 Cl(α)
    - 阻力系数 Cd(α)
```

### 3.2 叶素受力分析

在径向位置 r 处的叶素：

```
              来流方向
              ────→ V_local
                    ╲
                     ╲  α (迎角)
                      ╲
         ┌─────────────╲──┐
         │   翼型截面    ╲ │
         └────────────────╲┘
                          │
                          ├──→ dL (升力，垂直于来流)
                          └──→ dD (阻力，平行于来流)
```

局部来流速度：

$$
V_{local} = \sqrt{(\Omega r)^2 + (V_\infty + v_i)^2}
$$

局部迎角：

$$
\alpha = \theta_{blade} - \arctan\left(\frac{V_\infty + v_i}{\Omega r}\right)
$$

其中 θ_blade 为桨叶安装角（几何扭转角）。

### 3.3 积分得到总力

沿桨叶积分：

$$
T = N_b \int_0^R \frac{1}{2}\rho V_{local}^2 c(r) [C_l(\alpha)\cos\phi - C_d(\alpha)\sin\phi] dr
$$

$$
Q = N_b \int_0^R \frac{1}{2}\rho V_{local}^2 c(r) r [C_l(\alpha)\sin\phi + C_d(\alpha)\cos\phi] dr
$$

其中 φ 为来流角（inflow angle）。

## 4. 简化推力/扭矩模型

### 4.1 二次模型（最常用）

在大多数四旋翼论文和仿真中，使用简化的二次模型：

$$
T = k_T \cdot \omega^2
$$

$$
Q = k_Q \cdot \omega^2
$$

| 参数 | 含义 | 典型值范围 | 单位 |
|------|------|-----------|------|
| kT | 推力系数 | 1.0×10⁻⁵ ~ 5.0×10⁻⁵ | N/(rad/s)² |
| kQ | 扭矩系数 | 1.0×10⁻⁷ ~ 5.0×10⁻⁷ | N·m/(rad/s)² |
| ω | 电机转速 | 0 ~ 12000 | rpm（需转换为 rad/s） |

**推导依据**：

从量纲分析：
- T ∝ ρ·A·(ΩR)² = ρ·πR²·Ω²·R² = ρ·πR⁴·ω²
- 因此 kT ≈ C_T · ρ · π · R⁴

### 4.2 kT/kQ 的比值

$$
\frac{k_T}{k_Q} \approx \frac{C_T}{C_Q \cdot R}
$$

这个比值决定了：
- 给定推力下的反扭矩大小
- 偏航控制所需的转速差
- 旋翼效率（kT/kQ 越大越高效）

### 4.3 模型精度

| 飞行条件 | 二次模型误差 | 原因 |
|---------|-------------|------|
| 悬停 | < 5% | 设计工况 |
| 低速前飞 (< 3 m/s) | 5-10% | 来流不对称性小 |
| 中速前飞 (3-8 m/s) | 10-20% | 挥舞效应显著 |
| 高速前飞 (> 8 m/s) | > 20% | 需要 BEMT 或查表 |
| 大油门变化 | 10-15% | 动态入流滞后 |

## 5. 旋翼间干扰效应

### 5.1 尾流干扰

当一个旋翼处于另一个旋翼的尾流中时，其性能会下降：

```
    四旋翼前飞时的尾流分布（俯视）：

           前飞方向 →
        ┌─────────────┐
        │  2    ↗  4  │
        │      CG     │
        │  1    ↘  3  │
        └─────────────┘

    旋翼 1 的下洗流可能影响旋翼 3
    旋翼 2 的下洗流可能影响旋翼 4
```

| 干扰类型 | 影响 | 建模方法 |
|---------|------|---------|
| 垂直干扰 | 悬停时下方旋翼效率降低 | 降低 kT 系数 |
| 侧向干扰 | 前飞时尾流倾斜 | 推力矢量偏转 |
| 涡流干扰 | 大间距时影响小 | 通常忽略 |

### 5.2 间距比影响

旋翼间距与旋翼直径之比 d/D 对干扰的影响：

| d/D | 推力损失 | 适用场景 |
|-----|---------|---------|
| < 0.8 | 10-20% | 紧凑型无人机 |
| 0.8 - 1.0 | 5-10% | 标准四旋翼 |
| > 1.0 | < 5% | 大间距设计 |
| > 1.5 | 可忽略 | 几乎无干扰 |

> 大多数消费级四旋翼的 d/D 在 1.0-1.2 之间，干扰效应可忽略或通过略微降低 kT 来补偿。

## 6. 地面效应（Ground Effect）

### 6.1 物理机理

当旋翼接近地面时，下洗流被地面阻挡，桨盘下方压力升高，导致推力增加。

```
    远离地面：              接近地面：
    ┌───┐                  ┌───┐
    │ R │ → v_down         │ R │ → v_down
    └───┘                  └───┘
      ↓                      ↓
      ↓                     ═══════ 地面
      ↓                      → → → 气流偏转
    (无限制)               (压力升高，推力增加)
```

### 6.2 地面效应模型

推力增强因子：

$$
\frac{T_{GE}}{T_\infty} = \frac{1}{1 - (R/4h)^2}
$$

其中：
- T_GE = 地面效应下的推力
- T_∞ = 远离地面时的推力
- R = 旋翼半径
- h = 旋翼离地高度

| 高度比 h/R | 推力增强 | 说明 |
|-----------|---------|------|
| 0.5 | +33% | 极强地面效应 |
| 1.0 | +7% | 显著 |
| 1.5 | +3% | 轻微 |
| 2.0 | +1.5% | 很小 |
| > 3.0 | < 1% | 可忽略 |

### 6.3 地面效应对控制的影响

| 影响 | 描述 | 应对策略 |
|------|------|---------|
| 推力增加 | 相同转速产生更大推力 | 降低油门 |
| 效率提升 | 悬停功耗降低 | 自动适应 |
| 不对称效应 | 地面倾斜时各旋翼影响不同 | 需要补偿 |
| 着陆时振荡 | 推力-高度正反馈 | 降低控制增益 |

## 7. 前飞中的气动效应

### 7.1 前行桨叶与后行桨叶

前飞时，桨盘两侧的来流速度不对称：

```
    前飞方向 →

         前行桨叶              后行桨叶
    (V_tip + V_forward)    (V_tip - V_forward)
            ↑                    ↑
            │                    │
    ────────┼────────────────────┼──────── 桨盘
            │                    │
            ↓                    ↓
         升力较大              升力较小
```

### 7.2 挥舞运动

前行桨叶升力大 → 上挥 → 迎角减小 → 升力降低
后行桨叶升力小 → 下挥 → 迎角增大 → 升力增加

最终效果：桨盘前倾，产生低头力矩。这需要通过增加后旋翼推力来补偿。

### 7.3 前飞修正模型

前飞时的推力修正：

$$
T = k_T \cdot \omega^2 \cdot f(\mu)
$$

其中 μ = V/(ΩR) 为前进比，f(μ) 为修正因子。

简单修正：

$$
f(\mu) \approx 1 + k_\mu \cdot \mu^2
$$

## 8. 共轴旋翼配置

### 8.1 结构特点

```
         上旋翼 (CCW)
         ───────────
            │ 轴
         ───────────
         下旋翼 (CW)

    两个旋翼共轴反转
```

### 8.2 气动特性

| 特性 | 单层旋翼 | 共轴旋翼 |
|------|---------|---------|
| 上旋翼效率 | 100% | 100% |
| 下旋翼效率 | - | 60-80%（受上旋翼下洗流影响） |
| 总推力/面积 | 基准 | 1.6-1.8 倍 |
| 总扭矩 | 基准 | 接近零（反向旋转抵消） |
| 结构复杂度 | 低 | 高（需齿轮传动） |

### 8.3 共轴旋翼建模

下旋翼的推力修正：

$$
T_2 = k_T \cdot \omega_2^2 \cdot \eta_{coaxial}
$$

其中 η_coaxial ≈ 0.6 ~ 0.8，取决于旋翼间距和转速比。

## 9. 查表法 vs 解析模型

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| 二次模型 (kT·ω²) | 简单、计算快 | 精度有限 | 初步仿真、控制器设计 |
| BEMT 解析 | 有物理基础 | 计算复杂、需要翼型数据 | 精确仿真 |
| CFD 数值仿真 | 最精确 | 计算成本极高 | 桨叶设计 |
| 实验查表 | 直接可靠 | 需要实验设备 | 最终验证 |
| 混合方法 | 兼顾精度和效率 | 实现复杂 | 工程应用 |

```
│              │──→ Q (扭矩)
│ 行：ω        │

```

## 思考题

**1. 为什么二次推力模型 T = kT·ω² 在悬停时足够精确，但在前飞时误差增大？**

<details>
<summary>参考答案</summary>

二次模型基于以下假设：
1. 来流速度均匀且垂直于桨盘
2. 桨叶各处迎角相同
3. 无挥舞运动

悬停时这些假设基本成立：来流只有轴向诱导速度，分布较均匀。

前飞时违反了这些假设：
1. **来流不对称**：前行侧来流速度 = ΩR + V，后行侧 = ΩR - V
2. **迎角变化**：桨叶旋转一周内迎角周期性变化
3. **挥舞运动**：桨盘倾斜，推力矢量方向改变
4. **涡流效应**：桨尖涡随前飞速度倾斜，影响诱导速度分布

因此前飞时需要更复杂的模型（BEMT 或带修正因子的二次模型）来保持精度。
</details>

**2. 如何通过实验测量 kT 和 kQ？描述实验装置和步骤。**

<details>
<summary>参考答案</summary>

实验装置：
- 电子秤（精度 0.01g）或推力传感器
- 扭矩传感器（或通过电流反推）
- 电机驱动器（ESC）
- 转速传感器（光电式或基于反电动势）
- 数据采集系统（Arduino/STM32 + PC）

步骤：
1. 将电机固定在测试台上，测试台放置于电子秤上
2. 设置 ESC 输入为不同油门值（20% 到 100%，步长 5%）
3. 每个油门值等待转速稳定后记录：
   - 电子秤读数变化 Δm → T = Δm·g
   - 转速 n (rpm) → ω = n·2π/60
4. 绘制 T vs ω² 曲线
5. 线性拟合斜率即为 kT

类似地测量扭矩（用电流 I 和电压 V 估算 P = V·I，Q = P/ω），拟合得到 kQ。

注意事项：
- 消除气流干扰（在开阔空间测试）
- 多次测量取平均
- 检查桨叶是否平衡
</details>

**3. 地面效应在什么高度范围内需要建模？对自动着陆控制器有什么影响？**

<details>
<summary>参考答案</summary>

地面效应显著的高度范围：h < 2R（旋翼离地高度小于 2 倍旋翼半径）。

对于 R = 0.12m 的四旋翼，h < 0.24m（约 24cm）时需要考虑。

对自动着陆控制器的影响：

1. **推力高估**：控制器预期需要 T_hover 的推力来维持高度，但地面效应使其产生更大推力，导致飞行器被"推高"
2. **振荡风险**：接近地面 → 推力增加 → 上升 → 远离地面 → 推力恢复正常 → 下降 → 再次接近地面 → 循环
3. **解决方案**：
   - 在地面效应区域降低位置控制增益
   - 使用高度相关的推力补偿表
   - 着陆时切换到力控模式（而非位置控制）
   - 快速果断着陆，避免在地面效应区域悬停
</details>

**4. 共轴旋翼的下旋翼效率为什么只有上旋翼的 60-80%？能否通过调整间距改善？**

<details>
<summary>参考答案</summary>

下旋翼效率降低的原因：

1. **下洗流预加速**：上旋翼的下洗流已经加速了空气，下旋翼面对的是"已经被推动"的空气，诱导速度更大，根据动量理论，推力效率降低。

2. **湍流干扰**：上旋翼产生的尾流涡系直接冲击下旋翼，导致来流不均匀。

3. **桨尖涡干扰**：上旋翼的桨尖涡与下旋翼桨叶相互作用。

改善方法：
1. **增大间距**：间距越大，上旋翼尾流扩散越充分，影响越小。但会增加机体高度和重量。
2. **减小下旋翼转速**：降低下旋翼负荷比，使其在更高效的状态工作。
3. **桨叶设计**：下旋翼使用更高效的翼型和更大的弦长。
4. **倾斜桨盘**：将下旋翼略微倾斜，避开上旋翼核心尾流。

工程上，间距增大 30% 可将效率损失从 40% 降到约 20%，但需要权衡结构代价。
</details>

**5. 解释为什么旋翼的扭矩系数 kQ 比推力系数 kT 小约两个数量级。**

<details>
<summary>参考答案</summary>

这反映了旋翼的基本物理特性：产生推力的效率远高于产生扭矩。

量纲分析：
- T = kT·ω²，[kT] = N/(rad/s)²
- Q = kQ·ω²，[kQ] = N·m/(rad/s)²

kQ/kT 的量纲是 [m]，物理上对应一个"等效力臂"。

对于典型旋翼：
- kT ≈ 1.5×10⁻⁵ N/(rad/s)²
- kQ ≈ 2.0×10⁻⁷ N·m/(rad/s)²
- kQ/kT ≈ 0.013 m = 13 mm

这意味着扭矩相当于在 13mm 的力臂上施加推力。这个力臂远小于旋翼半径（~120mm），说明：

1. 反扭矩主要来自桨叶阻力（而非升力），阻力系数远小于升力系数（Cd/Cl ≈ 0.05-0.1）
2. 力臂是桨叶气动中心到旋转轴的距离的加权平均，有效力臂较小
3. 高效桨叶设计的目标就是最大化 kT/kQ 比值

这个比值也决定了偏航控制的效率：kT/kQ 越大，产生相同的偏航力矩需要的转速差越小。
</details>
