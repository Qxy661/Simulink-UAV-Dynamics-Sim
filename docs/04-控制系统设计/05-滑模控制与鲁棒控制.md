# 滑模控制与鲁棒控制

> 预计阅读：25 分钟 | 前置知识：非线性控制基础、Lyapunov 稳定性理论、状态空间模型

---

## 1. 鲁棒控制的动机

在实际无人机系统中，控制器面临以下挑战：

| 挑战 | 来源 | 影响 |
|---|---|---|
| 模型不确定性 | 简化建模、参数辨识误差 | 控制性能下降 |
| 外部扰动 | 风、气流、载荷变化 | 轨迹偏差 |
| 参数变化 | 电池电压下降、质量变化 | 稳定性问题 |
| 未建模动态 | 电机响应延迟、柔性结构 | 高频振荡 |

传统的 PID 和 LQR 控制器在面对上述问题时缺乏理论保证。**滑模控制（Sliding Mode Control, SMC）** 是一种鲁棒非线性控制方法，能在有界不确定性和扰动下保证系统稳定。

---

## 2. 滑模控制基本概念

### 2.1 核心思想

滑模控制的核心是设计一个**滑模面（Sliding Surface）**，使系统状态在到达滑模面后沿滑模面滑向平衡点。

```
状态空间中的滑模运动：

    ẋ₂ ↑
       │     ╱ s = 0 (滑模面)
       │   ╱
       │ ╱  ←── 到达阶段（Reaching）
       ╱
    ───╱───────► x₁
     ╱│         ←── 滑动阶段（Sliding）
   ╱  │
       │
```

### 2.2 控制结构

滑模控制律由两部分组成：

```
u = u_eq + u_sw

u_eq：等效控制（Equivalent Control）— 保持状态在滑模面上
u_sw：切换控制（Switching Control）— 将状态驱向滑模面
```

## 3. 滑模面设计

### 3.1 一阶滑模面

对于二阶系统：

```
误差：e = x_ref - x
滑模面：s = ė + λ·e    (λ > 0)
```

其中 λ 为滑模面斜率，决定滑动阶段的收敛速度。

当 s = 0 时，有 ė = -λe，误差按指数衰减。

### 3.2 高阶滑模面

对于 n 阶系统，使用更一般的滑模面：

```
s = (d/dt + λ)^(n-1) · e = e^(n-1) + cₙ₋₂·e^(n-2) + ... + c₁·ė + c₀·e
```

系数 cᵢ 通过 Hurwitz 多项式选取，保证滑动阶段的稳定性。

### 3.3 滑模面参数选择

| 参数 | 增大效果 | 减小效果 | 典型值 |
|---|---|---|---|
| λ | 收敛更快 | 收敛更慢 | 1~20 |
| 滑模面阶数 | 减小抖振 | 增加设计复杂度 | 1 或 2 |

## 4. 到达条件

### 4.1 到达条件（Reaching Condition）

系统状态到达滑模面的条件：

```
s · ṡ < 0
```

这意味着：
- 当 s > 0 时，ṡ < 0（s 减小）
- 当 s < 0 时，ṡ > 0（s 增大）
- 状态始终被驱向 s = 0

### 4.2 Lyapunov 到达条件

使用 Lyapunov 函数 V = s²/2：

```
V̇ = s·ṡ < 0    (当 s ≠ 0)
```

### 4.3 到达律设计

常用的到达律：

| 到达律 | 公式 | 特点 |
|---|---|---|
| 恒速到达 | ṡ = -η·sign(s) | 到达时间可预测 |
| 指数到达 | ṡ = -η·sign(s) - qs | 到达更快 |
| 幂次到达 | ṡ = -η·|s|^α·sign(s) | 减小抖振 |

## 5. 抖振问题与解决

### 5.1 抖振的产生

理想的滑模控制使用不连续的 sign(s) 函数，导致控制量在滑模面附近高频切换，产生**抖振（Chattering）**。

```
控制量 u
    │    ╱╲    ╱╲    ╱╲
    │   ╱  ╲  ╱  ╲  ╱  ╲   ← 抖振
────┼──╱────╲╱────╲╱────╲──► t
    │
```

### 5.2 边界层法（Boundary Layer）

用饱和函数 sat(s/Φ) 替代 sign(s)：

```
           ┌ sign(s)        if |s| > Φ
sat(s/Φ) = ┤
           └ s/Φ            if |s| ≤ Φ
```

其中 Φ 为边界层厚度。

| 方法 | 公式 | 抖振 | 鲁棒性 | 稳态精度 |
|---|---|---|---|---|
| sign(s) | 不连续切换 | 大 | 最强 | 理想滑模 |
| sat(s/Φ) | 连续近似 | 小 | 稍弱 | 边界层内 |
| tanh(s/Φ) | 光滑近似 | 最小 | 最弱 | 边界层内 |

### 5.3 高阶滑模控制（HOSM）

将不连续控制作用在控制量的高阶导数上，实际控制量变为连续的：

```
传统 SMC：u 不连续 → 输出抖振
HOSM：    u 连续，u̇ 不连续 → 输出平滑
```

## 6. 超螺旋算法（Super-Twisting Algorithm）

### 6.1 二阶滑模

超螺旋算法是一种二阶滑模控制方法，能产生连续的控制信号。

```
u = -k₁|s|^(1/2)·sign(s) + v
v̇ = -k₂·sign(s)
```

### 6.2 参数设计

增益需满足：

```
k₂ > Φ
k₁² ≥ 4Φ·(k₂ + Φ)/(k₂ - Φ)
```

其中 Φ 为扰动的上界。

### 6.3 超螺旋算法特性

| 特性 | 说明 |
|---|---|
| 控制连续性 | 控制信号 u 连续 |
| 有限时间收敛 | 状态在有限时间内到达滑模面 |
| 鲁棒性 | 对匹配不确定性鲁棒 |
| 无需 ṡ 信息 | 不需要 s 的导数 |

## 7. EKF 与滑模控制的结合

### 7.1 为什么要结合

滑模控制需要精确的状态信息（位置、速度、姿态、角速率）。实际中传感器存在噪声和延迟，使用 EKF（Extended Kalman Filter）进行状态估计可以提高控制性能。

```
传感器数据 ──► EKF ──► 状态估计 x̂ ──► SMC ──► 控制量 u
 (含噪声)      │                         │
               └── 状态协方差 P          └── 鲁棒控制
```

### 7.2 EKF 状态估计

EKF 用于估计四旋翼的完整状态：

| 估计状态 | 传感器来源 | EKF 作用 |
|---|---|---|
| 姿态 (φ, θ, ψ) | IMU (陀螺仪+加速度计) | 融合互补滤波 |
| 位置 (x, y, z) | GPS + 气压计 | 融合位置估计 |
| 速度 (vx, vy, vz) | IMU + GPS | 速度估计 |
| 角速率 (p, q, r) | 陀螺仪 | 去除偏置和噪声 |

### 7.3 集成架构

```
┌──────────────────────────────────────────────────┐
│              HSMC-EKF 控制架构                      │
│                                                    │
│  ┌──────┐   ┌─────┐   ┌─────────┐   ┌──────┐     │
│  │ IMU  │──►│ EKF │──►│  状态    │──►│ HSMC │──► u│
│  │ GPS  │──►│     │   │  x̂, P   │   │      │     │
│  │ 气压 │──►│     │   └─────────┘   └──────┘     │
│  └──────┘   └─────┘                                │
└──────────────────────────────────────────────────┘
```

## 8. 控制方法对比

### 8.1 PID vs SMC vs HOSM

| 特性 | PID | SMC | HOSM (Super-Twisting) |
|---|---|---|---|
| 设计复杂度 | 低 | 中 | 高 |
| 鲁棒性 | 弱 | 强 | 最强 |
| 抖振 | 无 | 大 | 无/极小 |
| 稳态精度 | 取决于积分 | 边界层内 | 高 |
| 计算量 | 最低 | 低 | 中 |
| 模型依赖 | 弱 | 中 | 中 |
| 有限时间收敛 | 无 | 有 | 有 |
| 处理扰动 | 被动 | 主动（有界） | 主动（有界） |
| 实现难度 | 低 | 中 | 高 |

## 9. 四旋翼位置滑模控制设计

### 9.1 系统模型

四旋翼简化位置动力学：

```
ẍ = g·tan(θ) + d_x
ÿ = -g·tan(φ) + d_y
z̈ = (T/m)·cos(φ)·cos(θ) - g + d_z
```

其中 d_x, d_y, d_z 为外部扰动（如风力）。

### 9.2 滑模面设计

以 x 轴为例：

```
误差：ex = x_ref - x
滑模面：sx = ėx + λx·ex = (ẋ_ref - ẋ) + λx·(x_ref - x)
```

### 9.3 滑模控制律

```
u_θ = (1/g)·[ẍ_ref + λx·ėx + ηx·sign(sx)]
```

类似地为 y 轴和 z 轴设计控制律。

### 9.4 Simulink 实现

```
┌────────┐   ┌───────────┐   ┌──────────┐   ┌──────────┐
│x_ref    │──►│ 滑模面计算 │──►│ 滑模控制律│──►│ 姿态转换  │──► θ_des
│ẋ_ref   │   │ s = ė+λe  │   │ u = u_eq │   │          │
│ẍ_ref   │   │           │   │   + u_sw  │   │          │
└────────┘   └───────────┘   └──────────┘   └──────────┘
```

## 10. Simulink 滑模控制实现

### 10.1 MATLAB Function Block 实现

```matlab
function u = smc_controller(x, x_ref, params)
    % 参数
    lambda = params.lambda;
    eta = params.eta;
    phi = params.boundary_layer;

    % 误差计算
    e = x_ref(1) - x(1);
    de = x_ref(2) - x(2);
    dde = x_ref(3) - x(3);

    % 滑模面
    s = de + lambda * e;

    % 等效控制
    u_eq = dde + lambda * de;

    % 切换控制（带边界层）
    if abs(s) > phi
        u_sw = eta * sign(s);
    else
        u_sw = eta * s / phi;
    end

    % 总控制
    u = u_eq + u_sw;
end
```

### 10.2 超螺旋算法实现

```matlab
function [u, v_out] = supertwist_controller(s, v_prev, params, dt)
    k1 = params.k1;
    k2 = params.k2;

    % 超螺旋控制律
    u = -k1 * sqrt(abs(s)) * sign(s) + v_prev;
    v_dot = -k2 * sign(s);

    % 积分更新
    v_out = v_prev + v_dot * dt;
end
```

    │ x_ref    │    │Controller│    │ 四旋翼   │    │   y      │

## 11. 风扰动下的四旋翼位置控制

### 11.1 风扰动模型

典型的 Dryden 或 Von Karman 风模型：

```
风速：V_wind = V_mean + V_turbulence

V_turbulence 的功率谱密度：
S(ω) = σ²·(2L/π)·1/(1 + (Lω)²)
```

| 参数 | 含义 | 典型值 |
|---|---|---|
| V_mean | 平均风速 | 0~15 m/s |
| σ | 湍流强度 | 1~5 m/s |
| L | 湍流尺度 | 50~500 m |

### 11.2 控制性能对比

在 5 m/s 平均风速、2 m/s 湍流强度下：

| 方法 | 位置误差 (RMS) | 最大偏差 | 稳态误差 |
|---|---|---|---|
| PID | 0.85 m | 2.3 m | 0.4 m |
| LQR | 0.52 m | 1.5 m | 0.2 m |
| SMC | 0.28 m | 0.8 m | < 0.05 m |
| HOSM | 0.18 m | 0.5 m | < 0.02 m |

## 思考题

**1.** 滑模控制中的 sign(s) 函数为什么会导致抖振？sat(s/Φ) 如何缓解这个问题？sat(s/Φ) 的边界层 Φ 应该如何选择？

<details>
<summary>参考答案</summary>

sign(s) 函数在 s = 0 处不连续（从 -1 跳到 +1），导致控制量在滑模面附近高频切换。实际系统中存在延迟和噪声，状态不会精确停留在 s = 0，而是在其附近快速穿越，产生抖振。

sat(s/Φ) 在 |s| ≤ Φ 范围内使用线性近似 s/Φ，使控制量连续变化，消除了不连续切换。

边界层 Φ 的选择：
- Φ 太小：接近 sign(s)，抖振未完全消除
- Φ 太大：鲁棒性下降，稳态精度降低
- 经验法则：Φ ≈ 0.01 ~ 0.1 × 系统状态的典型值
- 可以使用自适应边界层 Φ(t) 随时间或误差大小调整
</details>

**2.** 超螺旋算法的控制律 u = -k₁|s|^(1/2)·sign(s) + v 中，为什么使用 |s|^(1/2) 而不是 |s|？这对收敛速度有什么影响？

<details>
<summary>参考答案</summary>

使用 |s|^(1/2) 的原因：

1. **有限时间收敛**：|s|^(1/2)·sign(s) 的非线性特性保证状态在有限时间内到达 s = 0，而线性项只能保证渐近收敛
2. **连续性**：在 s = 0 附近，|s|^(1/2) 的导数为无穷大，但函数值本身连续且为零，保证控制信号 u 的连续性
3. **与积分项配合**：v 的积分项保证了对恒值扰动的补偿，|s|^(1/2) 项提供有限时间到达

收敛速度影响：
- 远离滑模面（|s| 大）时，|s|^(1/2) < |s|，收敛较慢
- 接近滑模面（|s| 小）时，|s|^(1/2) > |s|，收敛更快
- 整体上，有限时间收敛优于渐近收敛，特别是在 s 接近零时
</details>

**3.** EKF 的状态估计延迟对滑模控制性能有什么影响？如何在控制设计中补偿这个延迟？

<details>
<summary>参考答案</summary>

EKF 引入的延迟影响：
1. 控制器使用的状态估计 x̂(k) 对应的是 k-δ 时刻的真实状态
2. 延迟减小了有效相位裕度，可能导致振荡
3. 滑模控制的抖振抑制能力在延迟下减弱

补偿方法：
1. **预测补偿**：使用 EKF 的预测步（x̂(k|k-1)）而非更新步，提前一个采样周期
2. **增加边界层**：适当增大 sat(s/Φ) 的 Φ，增加稳定裕度
3. **降低控制增益**：η 取保守值，牺牲部分鲁棒性换取稳定性
4. **Smith 预估器**：在控制回路中加入延迟补偿器
5. **使用高阶 EKF**：提高估计精度，减小等效延迟
</details>

**4.** 滑模控制假设扰动有界（|d| ≤ Φ）。如果扰动超出这个界会发生什么？如何处理这种极端情况？

<details>
<summary>参考答案</summary>

当扰动超出设计界 Φ 时：
1. 到达条件 s·ṡ < 0 可能不满足，系统无法到达滑模面
2. 控制性能严重下降，误差可能发散
3. 系统仍可能保持稳定（取决于超出程度和系统本身的稳定性）

处理极端扰动的方法：
1. **增大增益**：增大 η 以覆盖更大的扰动范围，但会加剧抖振
2. **自适应增益**：在线估计扰动大小，自适应调整 η
3. **扰动观测器**：使用 DOB（Disturbance Observer）估计并补偿扰动
4. **鲁棒自适应 SMC**：结合自适应控制在线调整控制参数
5. **安全机制**：当误差超过阈值时切换到更保守的控制模式（如姿态恢复模式）
</details>

**5.** 对比 SMC 和 MPC 两种鲁棒控制方法在四旋翼应用中的优缺点。在什么场景下你会选择 SMC，什么场景下选择 MPC？

<details>
<summary>参考答案</summary>

**SMC 优点**：
- 对有界扰动具有强鲁棒性，理论保证明确
- 计算量低，适合嵌入式实时实现
- 不需要精确模型，对模型不确定性鲁棒

**SMC 缺点**：
- 抖振问题（虽然 HOSM 可以缓解）
- 约束处理能力弱
- 不能显式优化性能指标

**MPC 优点**：
- 天然处理约束（输入、状态、速率约束）
- 可以优化多步性能指标
- 可以预处理未来轨迹

**MPC 缺点**：
- 计算量大，实时性挑战
- 强依赖模型精度
- 参数调整不如 SMC 直观

**选择建议**：
- **选 SMC**：强风环境、模型不确定性大、嵌入式计算资源有限、对鲁棒性要求极高
- **选 MPC**：需要满足多重约束、轨迹跟踪精度要求高、有足够计算资源、约束满足是硬需求
</details>
